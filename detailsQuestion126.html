AirPlay [Air Play]? - A company's iPhone, iPad, MacBook, and TV are connected wirelessly to view screens, videos, and music on TV This function is to apply the screen mirroring (smart view) function applied to Android platform to Company A's products

  • □ TV support product - Samsung Smart TV released after 2018 □ Connection conditions - TV, company A device (iPhone/iPAD/Macbook) must be connected to the same network (Wi-Fi). □ Supportable firmware version - TV: ver. 1251.0 (2018 TV) / ver 1065.0 (2019 TV) or higher support - Mobile: Supports iSO 12.3 or higher / macOS
    10.14.5 or higher TV setting method □ Home → Settings → General → Apple Airplay Settings → Airplay On Remote control home button → Go to the lower left setting of TV screen General → Go to AirPlay setting Change AirPlay to On state Mobile setting method □ Enter control center IPhone X, iPad Pro: Enter by raising the screen up IPhone 6/7/8, etc.: Enter by lowering the screen □ How to set
    ① Select Screen Mirroring
    ② TV selection on the same network
    ③ Password display for T mobile connection
    ④ Enter and approve the password displayed on the TV
    ⑤ Normally connected to TV Troubleshooting
    1.AirPlay connection is not working - Check if the TV and mobile device exist on the same network - Check if it is turned on in TV AirPlay setting
    2. Home → Settings → General → No Airplay Settings - Check if it is the latest version of firmware version - ver. 1251.0 (2018 model) / ver 1065.0 (2019 model) or higher support - It occurs when the smart TV terms and conditions are not used, so it is necessary to check the smart TV terms and conditions.
