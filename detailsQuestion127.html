Please let me know how to set up the 2019 model integrated remote control.

Expected cause It may occur because the integrated remote control is not set

  • Method of action How to set up integrated remote control for 2019 model We support more than RU6990 model among 2019 products. The functions provided by TV are basic to change 'channel' and 'volume'.
    1.Remote control home key → external input
    2. Select external input → integrated remote control setting
    3. Add new device
    4. Set-top box (select external device to connect)
    5. Connect the set-top box to the TV when the TV is directly connected
    6. Area-of-residence
    7. Service provider setting (select cable company)
    8. Select the connected HDMI mode
    9. Set-top box control test (channel up / down)
    10. Select “Yes” if the channel has changed
    11. Setup complete. 12. Completion of my own smart hub Next choice 13. Remote control test volume button operation 14. Remote control test channel button operation 15. Watch TV - Select Initialization method If you return the setting to malfunction or shipment conditions, you can restore it to factory shipment conditions through initialization applied to the product. ■ Setting method : Press the home button of the remote control → Settings menu → Customer support → Self diagnosis → Initialization → Password 0000 → Initialization complete after rebooting
    1.Press the remote control home button
    2. Move [Settings] at the bottom left
    3. Select Settings Menu → Go to General [Initialization]
    4. Password [0000] Enter
    5. Select [Initialization]
    6. [Check] Reboot when selected and initialization completed
