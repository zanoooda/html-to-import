The 2019 smart TV moves too fast when you press the remote control button.

Symptom
1

  • If you press the remote control button on the smart TV, it moves too fast.
    2. The channel goes up by two spaces and down by two spaces.
    3. The speed of moving apps and menus is fast. cause When the remote control is operated, if you press the button for a long time, the signal is repeatedly recognized and the reaction speed can be felt as fast. Method of action
    1.Please proceed with the latest firmware upgrade. (ver 1303.7 or higher)
    2. Please adjust the response interval of the remote control. - When adjusting the remote control response interval, the set value is applied to the entire range of remote control operation. Detailed instructions for adjusting the remote control response interval. Remote Control Response Interval Adjustment Guide
    1.Home → Settings → General → Accessibility
    2. Repetitive setting of remote control button
    3. Response interval adjustment
    ※ You can set the button to slow down when you keep pressing the remote control button. You can adjust the operating speed at the response interval after setting the button repetition slow to turn on.
