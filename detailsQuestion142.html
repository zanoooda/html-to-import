Please tell me how to set the initial setting of The Sero TV.

Function guide Initial setup method of The Sero TV [Support Specifications]
※ The SERO can only be set with the SmartThings app on the smartphone


  • ※ Check the version of the smartphone and SmartThings app. - Smartphone version: Android
    6.0 or higher, iOS 10 or higher operating system - SmartThings app version: ver 1.7.33-26 or higher
    1.Turn on The Sero and run the SmartThings app on your smartphone.
    ※ Make sure your smartphone is connected to a wireless network (Wi-Fi).
    2. If the model is automatically searched, please select the 'Add Now' button to add The Sero model. If the model is not automatically searched, please search manually through 'Add device'.
    ※ You can add The sero model by running SmartThings app on your smartphone.
    3. Please choose the place and room where The Sero is installed.
    ※ If there is no place or room you want, you can add a new place or room.
    4. Consent to Smart Hub Terms of Use and Privacy Collection and Use is required. After that, please change the name of the TV you want.
    ※ You can use various services and useful functions through Samsung Smart TV.
    5. Back up my TV settings data to Samsung Cloud. You can simply restore the TV when initializing or replacing it.
    ※ Check if 'My Data Backup' is selected.
    6. Please check the connection status of the external device. Connect the cable to the TV for device recognition and then connect it to the external device Please make sure the power is on.
    ※ The image below sets the antenna cable connection, and please select the device according to the connection device you have.
    7. Set up artificial intelligence (AI) mode for a better viewing experience.
    ※ Artificial intelligence (AI) is only supported by QLED TVs in 2019 and may not be supported depending on the model.
    8. Enter a residential area and select a service provider to complete the setup.
    ※ When the channel search is completed, the TV setting is completed.
