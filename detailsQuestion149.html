The 2020 TV screen is blinking.

If the entire TV screen blinks, it is likely to be an abnormality in the broadcast signal or external device signal, or an abnormality in the cable that transmits the signal

  • Check and reconnect antenna cables If you are watching TV with an antenna, please completely separate and reconnect the antenna connection line between the TV and the wall terminal. HDMI Cable Test Method
    1.Select the Home button or Menu button → Settings → Customer Support → Device Care → Self Diagnosis → 'HDMI Problem Check' on the TV remote control.
    2. Please select the input terminal you want to test.
    3. If the connected screen looks fine, please select “Yes”.
    4. Adjust the volume. If you hear the sound of the connected device without any problems, please select “Yes.”
    5. If the cable is normal, no problem is found.
    6. But if you can't see the screen or hear the sound, choose No.
    7. Please choose to start the test.
    8. Depending on the condition of the cable, it may take 1 to 3 minutes, so please wait.
    9. When diagnosing HDMI cable problem, check the connection status and replace it with a genuine cable. Check and reconnect HDMI cable when using set-top
    1.I recommend you to use a genuine cable. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    2. Connect the set-top box and TV to check if the external input connection line such as HDMI is connected normally when watching the broadcast.
    3. If the same symptoms continue to occur, abnormalities in the broadcast signal or external device signal may occur. Please contact the cable company for a signal check.
