Please tell me how to delete the demo video from Samsung TV.

Method guide
※ Demo video? It is a video that is selected and played in store mode when displayed in the store

  • [Support Model] - 2016 UHD TV abnormality METHOD FOR DELETING DEMOTION IMAGE
    1.Press the home key on the remote control and go to Settings to select Simple Settings.
    2. Select Store Mode.
    ※ At this time, the usage mode must be set to store mode. In home mode, press the home key on the remote control, select [Settings], and change [System] → [Expert Settings] → Usage mode to [Buy Store Mode].
    3. If you select Delete Demo Content, it will be deleted.
