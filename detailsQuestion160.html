  • I can't connect to my PC with the TV HDMI cable. If the screen does not appear when connecting to the PC with the HDMI cable, there may be problems with the HDMI cable or PC setting. HDMI Cable Reconnection
    1.Connect the PC and TV to check if the HDMI cable is connected normally when using it. METHOD FOR CONNECTING PC
    1.HDMI cable is required for TV and PC connection. I recommend you to use a genuine cable.
    2. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    3. Please connect the HDMI cable to the PC and TV.
    4. Press the Home or Menu button on the TV remote control to select an external device connected in “External Input” and connect it to the PC. Resolution control method
    1.Right-click the window desktop.
    2. Select Display Settings.
    3. In Resolution, select Recommended Resolution.
    4. If you select Keep Changed Settings, PC resolution adjustment is completed. Frequency setting method
    1.Select Advanced Display Settings. Select “Show adapter properties”.
    2. Select all mode marks. Select the optimum frequency.
    3. Selecting the OK button completes the PC frequency setting.
    4. If the PC screen is displayed outside the TV screen, please set the external input editing to PC Please set the screen size to fit on the face.
    5. Please select Edit from “External Input” and “Connected External Input”.
    6. Please select the device icon by PC. METHOD FOR ADJUSTING SCREEN Size
    1.Press the Home or Menu button to select “On” for screen matching in “Settings,” “Screen,” and “Screen Size Settings.”
    2. When customizing the screen, adjust the screen by enlarging or reducing the image to fit the screen.
