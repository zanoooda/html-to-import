When using the TV set-top box in 2020, there is no voice or it is strange.

To watch the video, press the Play button (▶)

  • If there is no voice or something strange when using the set-top, it is likely to be an abnormality in the cable that transmits the signal or signal of the set-top. Sound quality test method
    1.Press the Home or Menu button on the TV remote control to select Settings → Customer Support → Device Care → Self Diagnosis → Sound Quality Test.
    2. If the reproduced sample sound is heard neatly without noise, it is a symptom of noise due to broadcast signal problems, not product problems.
    3. In this case, you should contact the broadcaster for a broadcast signal check. [wired, cable, IPTV, etc.]
    4. However, if there is noise in the reproduced sample sound, it is recommended to check the TV. Check and reconnect HDMI cable when using set-top
    1.I recommend you to use a genuine cable. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    2. Connect the set-top box and TV to check if external input connection lines such as HDMI are connected normally when watching the broadcast. Set-top power checking method
    1.If there is no set-top power light, please reconnect the power. When the set-top power is on standby, use the set-top remote control to turn on the power.
    2. If the power is not turned on, please contact the set-top manufacturer (communication company). Set-top volume control
    1.Set-top volume control can be performed by pressing the +, - buttons on the set-top remote control to increase or decrease the set-top volume.
    2. If you adjust the volume with the TV remote control on a normal basis, set the set-top volume to medium.
