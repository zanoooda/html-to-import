When using the TV set-top box in 2020, the power is turned off or on intermittently.

If the power is turned off or on intermittently when using the set-top, it is caused by an abnormal signal on the set-top

  • Event log checking method
    1.Press the Home or Menu button on the TV remote control to select Settings → Customer Support → Customer Support Information.
    2. Please use the direction button of the remote control on the customer support information screen and pass it to the next page.
    3. If there is information in the event log, the TV is operated by an external device. If it coincides with the intermittent occurrence date, please check the external device.
    4. If there is no event log information at the bottom of the customer support information, it is not a malfunction caused by an external device, so please update the TV to the latest version. How to disable TV Anynet+ (HDMI-CEC)
    1.Press the home button on the TV remote control to set → General → Select Anynet + (HDMI-CEC) in external device management and change it to Release. Check and reconnect HDMI cable when using set-top
    1.I recommend you to use a genuine cable. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    2. Connect the set-top box and TV to check if external input connection lines such as HDMI are connected normally when watching the broadcast. Software update
    1.Click Home or Menu on the TV remote control to set → Customer Support → Software Update → Select Update Now.
    2. Do not turn off the TV during the update. When the update is complete, the TV turns off and on automatically. At this time, both screen and sound settings can be initialized, so it is recommended to take notes of the settings in advance.
    3. Please proceed with the update when you find the latest version.
