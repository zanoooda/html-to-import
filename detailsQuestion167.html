The input mode changes automatically while watching TV in 2020.

[Video subtitles] If the input signal changes while watching TV, it occurs due to the error signal of the external device

  • Event log checking method
    1.Press the Home or Menu button on the TV remote control to select Settings → Customer Support → Customer Support Information.
    2. Please use the direction button of the remote control on the customer support information screen and pass it to the next page.
    3. If there is information in the event log, the TV is operated by an external device. If it coincides with the intermittent occurrence date, please check the external device.
    4. If there is no event log information at the bottom of the customer support information, it is not a malfunction caused by an external device, so please update the TV to the latest version. TV initialization method
    1.Press the Home or Menu button on the TV remote control to select Settings → General → Initialization.
    2. * The network password will be deleted during initialization, so please check (memo) the password before execution.
    3. Please enter your PIN number. The initial PIN number is 0000.
    4. Please select "Initialize" to proceed with initialization. Initialization proceeds.
    5. When initialization was completed, the TV power was automatically turned off. It turns on again. How to disable TV Anynet+ (HDMI-CEC)
    1.Press the home button on the TV remote control to set → General → Select Anynet + (HDMI-CEC) in external device management and change it to Release. Software update
    1.Click Home or Menu on the TV remote control to set → Customer Support → Software Update → Select Update Now.
    2. Do not turn off the TV during the update. When the update is complete, the TV turns off and on automatically. At this time, both screen and sound settings can be initialized, so it is recommended to take notes of the settings in advance.
    3. Please proceed with the update when you find the latest version.
