The corners of the 2020 TV screen look dark or bright.

To watch the video, press the Play button (▶)


  • ※ If the screen is blurred when using the set-top, it is highly likely that the resolution setting of the set-top is set to low resolution or abnormality of the set-top network. Image quality test method
    1.Click the home or menu of the TV remote control to set → Customer support → Device care → Self-diagnosis → Image quality test.
    2. Please select the "OK" button on the image quality test screen. The test photo will be displayed soon.
    3. If the test screen appears normally, select "Yes"
    4. If the diagnosis result is normal, it is likely to be a signal from an external device or a cable abnormality. Please check the external device or the connection status.
    5. If the image quality test screen does not appear normally, select "No". Additional test screen appears.
    6. If you have the same symptoms in the additional sample image, you'd better check the TV. Check and reconnect HDMI cable when using set-top
    1.I recommend you to use a genuine cable. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    2. Connect the set-top box and TV to check if external input connection lines such as HDMI are connected normally when watching the broadcast. Set-top resolution setting method
    1.Press the "Home" button on the set-top remote control to enter the set-top menu.
    2. Please select "MY" from the top menu.
    3. Go to the bottom and select Settings → Screen / Sound Settings.
    4. Please set it according to the resolution you use. If you don't know the resolution well, choose "Automatic". For more information on how to set up, please contact each carrier (settop). Set-top power checking method
    1.If there is no set-top power light, please reconnect the power. When the set-top power is on standby, use the set-top remote control to turn on the power.
