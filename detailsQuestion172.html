I want to operate the set-top box with the TV remote control in 2020.

[Video subtitles] If the integrated remote control is not set with the TV remote control or malfunctions, there is an obstacle or misset between products

  • Integrated remote control setting method
    1.First, check if the device you want to set up and the smart remote control are connected to the TV.
    2. Press the home button on the TV remote control to select External Input → Integrated Remote Control Settings.
    3. Please check if the device is powered on and press the start button.
    4. Please select the type of device to set the integrated remote control.
    5. If the set-top box you want to set is directly connected to the TV, please select "Connect directly to the TV."
    6. If the set-top box you want to set is connected to the TV through the home theater, please select "Connect through the home theater".
    7. Enter "Resident Area" to find a faster service provider and set up your TV.
    8. Set up a broadcasting service provider to experience various broadcasts. Select the external input terminal of the TV to which the set-top box you want to set is connected.
    9. Use a simple test to see if you can operate the set-top box.
    10. Select the "Channel Up" and "Channel Down" buttons on the screen and check the right screen.
    11. If the channel has changed, select the "Yes" button, and if the channel has not changed, select the "No" button. 12. Settings are complete. Now try all the devices connected with one Samsung smart remote control. Precautions for Installing Integrated Remote Control
    1.If there is a blockage due to an obstacle between products when installing the integrated remote control, a reception failure may cause a failure in installing the integrated remote control.
    2. Remove obstacles between products and proceed with the installation of integrated remote control.
    ※ If the set-top box does not work with the TV remote control Please check if the set-top box is located near the IR reception part of the TV.
    ※ If the one-connect box is a connected TV product Please use it for the set-top box near the one-connect box IR output.
