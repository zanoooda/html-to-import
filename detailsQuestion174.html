I can't connect the TV to the set-top box in 2020.

To watch the video, press the Play button (▶)

  • If the set-top box is not recognized or connected, it is likely to be more than a cable that transmits signals or signals from the set-top. External input switching method
    1.Press the home button on the TV remote control to select external input → set top (HDMI1).
    2. Press the home button on the TV remote control to enter externally → Select TV.
    3. Press the home button on the TV remote control to select external input → game console (HDMI4). HDMI connection terminals may vary by TV model. Check and reconnect HDMI cable when using set-top
    1.I recommend you to use a genuine cable. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    2. Connect the set-top box and TV to check if external input connection lines such as HDMI are connected normally when watching the broadcast. HDMI Cable Test Method
    1.Press the home or menu button on the TV remote control to set → Customer support → Device care → Self-diagnosis → HDMI problem check.
    2. Please select the input terminal you want to test.
    3. If the connected screen looks fine, please select "Yes".
    4. Adjust the volume. If you hear the sound of the connected device without any problems, please select "Yes".
    5. If the cable is normal, no problem is found. But if you can't see the screen or hear the sound, choose "No."
    6. Please choose to start the test. Depending on the condition of the cable, it may take 1 to 3 minutes, so please wait.
