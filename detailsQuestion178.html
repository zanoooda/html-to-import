41 METHOD FOR CONNECTING TV AND PC/Notebook WITH HDMI CABLE If the screen does not appear when connecting the PC with the HDMI cable, there may be problems with the HDMI cable or PC setting

  • HDMI Cable Reconnection Connect the PC and TV to check if the HDMI cable is connected normally when using it. METHOD FOR CONNECTING PC HDMI cable is required for TV and PC connection. We recommend using genuine cables. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur. Please connect the HDMI cable to the PC and TV. Press the home button on the TV remote control to select a PC from the external input and connect it. Resolution control method
    1.Right-click on the window screen. Select Display Settings.
    2. Select the recommended resolution (1920*1080) from the resolution.
    3. If you select Keep Changed Settings, PC resolution adjustment is completed. Frequency setting method
    1.Select Advanced Display Settings.
    2. Select Show Adapter Properties.
    3. Select Show All Modes.
    4. Select the optimum frequency.
    5. If you select the OK button, the PC setting is completed. If the PC screen is displayed outside the TV screen, please set the external input editing to PC and set the screen size to match the screen. METHOD FOR ADJUSTING SCREEN Size
    1.Press the Home or Menu button on the TV remote control to select Settings → Screen → Screen Size Settings → Screen Fit On.
    2. When customizing the screen, adjust the screen by enlarging or reducing the image to fit the screen. Switch to TV screen with Samsung laptop shortcut key
    ※ If the screen of the laptop does not appear even though the resolution is adjusted, frequency setting, and screen size are adjusted on the laptop In the case of Samsung laptops, the laptop and TV are connected by HDMI and the bottom left of the keyboard FN key Press the button. F4 key Please try to switch the screen (TV ↔ notebook screen) by pressing once. If the laptop screen appears on the TV when switching screens, it is in a normal state. If the above method does not solve the problem, please contact the laptop manufacturer. METHOD FOR SELECTING AN EXTERNAL INPUT DEVICE FOR A TV
