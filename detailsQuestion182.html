I can see vertical lines on the 2020 TV screen.

To watch the video, press the Play button (▶)

  • If a vertical line (vertical line) is visible on the TV screen, there is something wrong with the TV, there is a problem with the broadcast signal, and if it occurs at the top or bottom of the screen, it is a signal characteristic of an external device. Image quality test method
    1.Click the home or menu of the TV remote control to set → Customer support → Device care → Self-diagnosis → Image quality test.
    2. Please select the "OK" button on the image quality test screen. The test photo will be displayed soon.
    3. If the test screen appears normally, select "Yes"
    4. If the diagnosis result is normal, it is likely to be a signal from an external device or a cable abnormality. Please check the external device or the connection status.
    5. If the image quality test screen does not appear normally, select "No". Additional test screen appears.
    6. If you have the same symptoms in the additional sample image, you'd better check the TV. Check and reconnect antenna cables
    1.If you are watching TV with an antenna, please completely separate and reconnect the antenna connection line between the TV and the wall terminal. Automatic channel setting method
    1.Click Home or Menu on the TV remote control to select Settings → Channel / Broadcasting → Automatic Channel Settings.
    2. The number of channels searched may vary depending on the region, broadcaster, and surrounding environment.
    3. If the channel or automatic channel setting is disabled, we are currently using an external device (set-top box), so please change the external input to 'TV broadcasting'. METHOD FOR ADJUSTING SCREEN Size
    1.Press the Home or Menu button on the TV remote control to select Settings → Screen → Screen Size Settings → Screen Size.
    2. If the screen size is set to custom, the user can zoom in and adjust the location.
    3. Please increase the size horizontally so that the vertical line is not visible.
