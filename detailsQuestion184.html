    1.Click Home or Menu on the TV remote control to select Settings → Channel / Broadcasting → Automatic Channel Settings.
    2. The number of channels searched may vary depending on the region, broadcaster, and surrounding environment.
    3. If the channel or automatic channel setting is disabled, we are currently using an external device (set-top box), so please change the external input to 'TV broadcasting'. Check and reconnect HDMI cable when using set-top
    1.I recommend you to use a genuine cable. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
