Question content There are no buttons on the TV other than the remote control, is it not there originally? Do I have to operate it only with a remote control or if I can't find it, can't I watch TV if I lose the remote control or break down? Answers Of the UN40H5200AF model There are no buttons on the front of the TV TV Use the panel key on the side to use external input and TV power on/off on the "Function Menu" screen I can do it

  • Please refer to the image below for detailed usage. Using panel keys
    1.Power On: Press the panel key when the TV is on standby to turn on the TV.
    2. Adjust volume: Adjust volume by moving the panel key left and right while the TV is on.
    3. Select a channel: With the TV turned on, move the panel key up and down to select a channel.
    4. Use function menu: Press the panel key with the TV on to display the function menu screen. If you press the panel key again, the function menu screen disappears.
    5. Using Multimedia: Select Multimedia by moving the panel key on the feature menu screen. The Media Play screen appears.
    6. Using Menu: Select 'Menu' by moving the panel key on the Functional Menu screen. The menu screen appears. It is the same as the menu screen that appears when you press the 'Menu' button on the remote control.
    7. Select TV/External Input: Move the panel key on the Function menu screen to select TV/External Input. The TV/External Input Screen appears.
    8. Power off: On the function menu screen, select Power by moving the panel key. Press the panel key for at least 1 second to exit from menu, media play, and TV/external input. If you lose the remote control, you can move or select each item in the menu with the panel key at the bottom of the TV. If there is no input signal, the TV automatically switches to standby mode after 15 minutes. When operating the desired function by moving the panel key up, down, left, and right, be careful not to press the panel key. If you press the panel key first, you cannot move up, down, left, right.
