The 2020 TV screen is blurred or the picture quality is bad.

To watch the video, press the Play button (▶)

  • The case where the screen is blurred on the TV may occur when the resolution of the broadcast signal, the content, or the set-top box is low. Broadcasting signal test method
    1.Press the home or menu button on the TV remote control to select Settings → Customer Support → Device Care → Self Diagnosis → Signal Information.
    2. If the antenna signal is 4 or less or the SNR value is 20 or less, it is recommended to check the broadcast signal.
    3. Signal information is a feature only available on digital broadcast channels. Antenna pin status checking method
    1.Please check the status of antenna cable pin.
    2. If you are using a distributor or booster, please completely separate and reconnect it. TV channel check
    1.Please change the channel to the TV remote control and check if the channel notation is displayed as 9-1, 10-1, 11-1 on the screen.
    2. If it is changed to 9, 10, 11 without the “-“ mark, it is an analog broadcast, not a digital broadcast.
    3. In this case, manually press any digital channel such as ”9-1”, “10-1”, “11-1” to check. Automatic channel setting method
    1.Click Home or Menu on the TV remote control to select Settings → Channel / Broadcasting → Automatic Channel Settings.
    2. The number of channels searched may vary depending on the region, broadcaster, and surrounding environment.
    3. If the channel or automatic channel setting is inactive, please change the external input to 'TV broadcasting' because you are currently using an external device (set-top box). Set-top resolution setting method
    1.Press the “Home” button on the set-top remote control to enter the set-top menu.
    2. Please select “MY” from the top menu.
    3. Go to the bottom and select Settings → Screen / Sound Settings.
    4. Please set it according to the resolution you use. If you don't know the resolution well, choose “Auto”.
    5. For more information on how to set up, please contact each carrier (settop).
