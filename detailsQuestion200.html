To watch the video, press the Play button (▶)

  • If you want to restore the value you set when using the TV, or if the TV stops working or shows no response, you can initialize the TV and use it. TV power checking method
    1.Please completely disconnect and reconnect the one-connect power cord and wall power cord.
    2. Please completely disconnect and reconnect the TV power cord and wall power cord. TV initialization method
    1.Please select Home button or Menu button → Settings → General → Initialization of TV remote control.
    2. Please enter your PIN number. The initial PIN number is 0000.
    3. Please select “Initialize” to proceed with initialization. Initialization proceeds.
    4. When initialization is completed, the TV power is automatically turned off and then turned on again. Screen initialization method
    1.Select the Home button or Menu button → Settings → Screen → Expert Settings → TV Screen Initialization on the TV remote control.
    2. Please select “Initialize” to initialize the screen settings. Acoustic initialization method
    1.Please select the home button or menu button → setting → sound → expert setting → 'sound initialization' of the TV remote control. Smart hub initialization method
    1.Enter the home button or menu button → setting → customer support → device care → self-diagnosis → smart hub initialization of the TV remote control and proceed with initialization.
    2. Please enter your PIN number. The initial PIN number is 0000.
