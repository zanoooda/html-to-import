Usage guide [2016 Integrated Remote Control Support Model] - Support model: KU7000 or higher - Usage environment: The integrated remote control function provided by the TV changes 'channel' and 'volume', and power and other functions are basically not supported - How to set: Remote control home key ( ) → External input → Integrated remote control setting ■ 2016 Integrated remote control set-top box setting Screen → System → Integrated remote control setting (You can select from the menu after pressing external input)
1

  • Remote control home key → external input
    2. Select external input → integrated remote control setting
    3. Add new device
    4. Set-top box (Select external device to connect)
    5. Directly connect to the TV
    6. Area-of-residence
    7. Service provider setting (select cable company)
    8. Select the connected HDMI mode
    9. Set-top box control test
    10. Select “Yes” if the channel has changed
    11. Setup complete. ■ Initialization setting method Press the home button of the remote control → Settings → Customer Support → Self Diagnosis → Initialization → "0000" → "Yes" However, if it is not a smart remote control, you only need to press the menu button Press the remote control home button Press Settings on the Screen Click Customer Support Self-diagnosis press Initialization press Enter the number "0000" and press "Yes"
