The screen doesn't come out after connecting the camcorder / game console.

Expected cause Make sure that the external input is switched to HDMI (mode of the terminal connected to the game console or camcorder) on the TV

  • Check if cables (HDMI, components, etc.) are connected to camcorders, game consoles, and TVs. Method of action Check TV/External input switching Switch to HDMI mode using the TV remote control. Switching TV/External Input Press the TV/External Input button on the TV remote control Switch to HDMI 2 connected to the game console. Check the operation of the game console Check if the camcorder or game console is playing with normal power on and if the resolution setting of the camcorder or game console is normal. The verification method varies from device to device. Check after connecting cable Check cable connection Connect the HDMI (ARC terminal) cable to the TV. HDMI cable HDMI cables on game consoles.
