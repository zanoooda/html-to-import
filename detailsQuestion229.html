Please tell me how to set my picture on Samsung Smart TV Magic Screen.

Function guide How to Set My Photo on Smart TV Magic Screen
※ Only smart TVs in 2018 and 2019 are applicable

  • It can only be set in the SmartThings app on mobile.. In order to set up my personal photo on the magic screen, the TV must be registered in the SmartThings app on the mobile.
    1.Please check if the SmartThings app is installed on your mobile device and connect to the same network (Wi-Fi) as the TV.
    2. Run the SmartThings app on your mobile and select a TV.
    3. Please choose Magic Screen.
    4. In [My Album], select the 'Select Photos' menu and select Photos.
    5. Once the photo selection is completed, you can select View on TV to view the photos you want on the big screen of the TV.
    6. In addition to my photos, you can choose Deco, Art, etc. to decorate the TV screen according to the space.
    7. Select the menu you want, such as Special Edition, Deco, Information, etc. Please choose ‘Watch on TV.’
