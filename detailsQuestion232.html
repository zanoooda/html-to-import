I can't get a specific channel voice.

Expected cause It can be caused by a set-top box or a broadcast signal abnormality

  • Method of action Broadcast signal sensitivity check Symptoms of no voice on a specific channel can occur due to broadcasting signals or TV abnormalities, and the function of checking the sensitivity of broadcasting signals built into the TV can check the abnormality/nothingness of the product. [Setting Method] Press the home button of the remote control → Settings → Menu → Customer Support → Self-diagnosis → Start signal information Customer Support - Self Diagnosis - Start Signal Information Check the signal information for the channel while watching. If there are less than 4 antenna signals or the SNR value is less than 20, it is recommended to check the broadcast signal in this case if the broadcast signal is weak. Sound quality test confirmation Symptoms of no voice on a specific channel can occur due to broadcasting signals or TV abnormalities, and the sound quality test function can check for abnormalities in the product. [Setting Method] Menu → Customer Support → Self Diagnosis → Sound Quality Test Press the home button of the remote control Move [Settings] at the bottom left Menu → Customer support → Move self-diagnosis Select [Start sound quality test] During the sound quality test, the sample sound built into the TV is played, and if the sample sound played is heard neatly without noise, it is a symptom of noise due to broadcasting signal problems, not product problems. In this case, please contact the broadcaster to check the broadcast signal [wired, cable, skylife, IPTV, etc.]. If noise is generated in the reproduced sample sound, it can be caused by an abnormality in the speaker or voice circuit and must be checked. Check the latest version update Software malfunctions may prevent certain channel voices from being heard, which requires upgrades to the latest version of the software. Even after the upgrade, if it is the same, please contact the broadcaster for a broadcast signal check [wired, cable, skylife, IPTV, etc.]
    ① Internet Upgrade - Before 2012: Menu → Customer Support → Software Update → Internet Up - After 2013: Menu → Customer Support → Software Update → Update now Menu → System Software update Update now
    ② USB Upgrade Downpath: Access to Samsung Electronics homepage → Customer support → Model search → Download → Firmware → Download firmware file to PC → Decompress the received firmware file and save it in USB memory → Connect USB to TV [Installation Method] - After 2013: Menu → Customer Support → Software Update → Update now - Before 2013: Menu → Customer Support → Software Upgrade → USB Upgrade Menu - Customer Support Software upgrade Select Upgrade to USB
