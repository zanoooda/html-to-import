Find out how to use multi-link screen in curved TV UHD 9000 model series

UHD 9000 Model Link Screen (Screen Split Function) Execution Method While watching TV, you can use various functions such as web search and apps at the same time

  • Press the Menu button to select Run Screen > Multilink Screen (Screen Split Function).
    1.Press the multi-screen button below Samsung Smart Control (Integrated Remote Control).
    2. Select the screen you want using Samsung Smart Control (integrated remote control), watch TV, search the web, or use the app. - If you press the multi-screen button, you can select a function supported by the multi-link screen (screen split function).
    ※ Before using the multi-link screen (screen split function), check if it is connected to the Internet first Do it. To use a multi-link screen, you must be connected to the Internet. Before using the smart hub with the multi-link screen (screen split function), On TV Settings (Smart Hub > On TV > On TV Settings) Complete the smart hub setting with. METHOD FOR USING MULTILINK SCREEN (Screen Split Function) Content and channels of UHD or FHD resolution are not available at the same time. Using multilink screen, Smart hub When you run, the TV automatically shuts down the multi-link screen (screen split function) and runs the smart hub. Some TV features or apps may not be supported on the multi-link screen (screen split feature). In this case, exit the multi-link screen (screen split function) and use the function. 1 Select a feature to use
    1.While running the multi-link screen (screen split function), press the multi-screen button of Samsung Smart Control (integrated remote control). The Options menu appears on the screen.
    2. Select the features you want from the Options menu. Execute the selected feature.
    ※ The function supported by multi-link screen (screen split function) varies depending on the country. 2 Changing Channels or Volumes
    1.Move the pointer or focus to the screen where you want to change the channel or volume.
    2. Press the channel button to change the channel and the volume button to change the volume. You can change the channel or volume of the selected screen. 3 Connecting Bluetooth Headphones If you connect your Bluetooth headphones to your TV, you can listen to other screens with your Bluetooth headphones. Compatibility problems may occur depending on the type of device. (Mobile-only headphones may not work. ) Depending on the type of device, lip-synching may occur in which the image and voice do not match. Disconnection may occur depending on the distance between the TV and headphones. Bluetooth headphones and SoundShare functions are not available at the same time.
    1.First, put your Bluetooth headphones on hold. Please refer to the manual of Bluetooth headphones to make the connection standby.
    2. While running the multi-link screen (screen split function), press the multi-screen button of Samsung Smart Control (integrated remote control). The Options menu appears on the screen.
    3. Select Settings from the Options menu.
    4. Select Bluetooth headphone connection from the Settings menu. The Bluetooth headphone connection screen appears on the screen and automatically searches for connectable headphones. - If you can not search, place your Bluetooth headphones close to the TV and select Research. Re-search for paired headphones.
    5. Select the Bluetooth headphones you want to connect from the search-completed list, and then select Pairing and Connection. The TV and Bluetooth headphones are connected. 4 Select the sound of the screen you want to hear You can select and listen to the sound of the screen you want to listen to on the TV speaker or Bluetooth headphones, respectively.
    1.During multi-link screen execution (screen division function), press the multi-screen button of Samsung Smart Control (integrated remote control). The Options menu appears on the screen.
    2. Select Settings from Options.
    3. Select Speaker Settings from the Settings menu.
    4. Select the screen you want to hear from the TV speaker and Bluetooth headphones. TV speakers and Bluetooth headphones cannot be selected as sound on the same screen. Automatic only supports TV speakers. Bluetooth headphones are not supported.
