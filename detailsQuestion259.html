  • When using the set-top box, the screen is not blurred or clear.
    ※ If the screen blinks or looks broken when using the set-top, it is highly likely to be an abnormality of the cable that transmits the signal or signal of the set-top. Screen test method
    1.Click Home or Menu on the TV remote control to select “Settings,” “Customer Support,” “Self Diagnosis,” and “Start Screen Test.”
    2. Please “check” the image quality test notification message. Sample images for testing are displayed.
    3. “Do you still see the symptoms of the problem in the test video? The message ” appears.
    4. If the image quality test screen is normal, select “No”.
    5. Please check the broadcast signal rather than the TV problem.
    6. If you have the same symptoms in the additional sample image, you'd better check the TV. Check and reconnect HDMI cables when using set-top
    1.We recommend using genuine cables. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    2. Please check the genuine logo. Check the print or imprint of “Ultra High Speed HDMI” on the HDMI cable.
    3. Connect the set-top box and TV to check if external input connection lines such as HDMI are connected normally when watching the broadcast. Set-top resolution setting method
    1.Press the “Home” button on the set-top remote control to enter the set-top menu. Please select “MY” from the top menu.
    2. Go to the bottom and select “Settings” and “Screen/Sound Settings”. Please set it according to the resolution you use.
    3. If you don't know the resolution well, choose “Auto”. For more information on how to set it up, please contact each carrier. Set-top power checking method
    1.If there is no set-top power light, please reconnect the power.
    2. When the set-top power is on standby, use the set-top remote control to turn on the power.
    3. If the power is not turned on, please contact the set-top manufacturer. How to check router (AP) power
    1.If there is no router power light, please reconnect the power.
    2. Please check if there is a light on the router power source.
    3. If the power is not turned on, please contact the router manufacturer
