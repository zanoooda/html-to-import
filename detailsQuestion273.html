  • The input mode changes automatically while watching TV.
    ※ If the input signal is changed while watching TV, it occurs due to the external device error signal. Event log checking method
    1.Press the Home or Menu button on the TV remote control to select “Settings,” “Customer Support,” and “Customer Support Information.”
    2. The event log function will be applied from the 2019 model.
    3. Please use the direction button of the remote control on the customer support information screen and pass it to the next page.
    4. If there is information in the event log, the TV is operated by an external device.
    5. If it coincides with the day of intermittent occurrence, please check the external device.
    6. If there is no event log information at the bottom of the customer support information, it is not a malfunction caused by an external device, so please update the TV to the latest version. TV initialization method
    1.Press the Home or Menu button to select "Settings", "General", and "Initialization".
    2. Please enter the pin number. The initial pin is 0000. Please select the "Yes" button to proceed with initialization. Initialization proceeds.
    3. When initialization was completed, the TV power was automatically turned off. It turns on again.
    4. The network password will be deleted when the TV is initialized, so please check the password before execution. Anynet setting method
    1.Press the Home or Menu button on the TV remote control to select "Anynet Plus" from "Settings", "General", and "External Device Management" and change it to OFF. Software update
    1.Click Home or Menu on the TV remote control to select Update now in “Settings,” “Customer Support,” and “Software Update.”
    2. Do not turn off the TV during the update.
    3. When the update is complete, the TV turns off and on automatically.
    4. At this time, both screen and sound settings can be initialized, so it is recommended to take notes of the settings in advance.
    5. Please proceed with the update when you find the latest version.
