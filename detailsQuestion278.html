  • I can't search with TV voice recognition.
    ※ If the TV voice recognition function does not work, it may occur depending on the usage environment, network status, and voice recognition server status Network status checking method
    1.If the network connection status is “not connected,” you can check the symptoms through the network status. Please check the symptoms and take action.
    2. Network settings allow you to run network settings immediately. METHOD FOR USING VOICE RECOGNITION
    1.You can perform voice commands by pressing the remote control microphone button.
    2. For example, you can say “broadcast guide” to run the broadcast guide.
    3. If the “text statement” does not appear on the screen when executing the voice command, please check the network connection status that the network is not connected. How to check router (AP) power
    1.If there is no router power light, please reconnect the power.
    2. Please check if there is a light on the router power. .
    3. If the power is not turned on, contact the router manufacturer /li> Network reset
    1.The network password will be deleted when network initialization is executed, so please check the password before execution. Wireless network reset method
    1.Press the Home or Menu button to select Wireless in “Settings,” “General,” “Network,” and “Run Network Settings.”
    2. Check and connect the set network name and password of the wireless router.
    3. The network name and password can be found on the settings screen of the wireless router. Wired network reset method
    1.Press the Home or Menu button to select Wired in “Settings,” “General,” “Network,” and “Run Network Settings.”
    2. Check and connect the set network name and password of the wireless router.
    3. The network name and password can be found on the settings screen of the wireless router. Software update
    1.Click Home or Menu on the TV remote control to select Update now in “Settings,” “Customer Support,” and “Software Update.”
    2. Do not turn off the TV during the update.
    3. When the update is complete, the TV turns off and on automatically.
    4. At this time, both screen and sound settings can be initialized, so it is recommended to take notes of the settings in advance.
    5. Please proceed with the update when you find the latest version.
