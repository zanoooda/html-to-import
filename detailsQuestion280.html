I can't connect to the set-top box

  • If the set-top box is not recognized or connected, it is likely to be more than a cable that transmits a set-top signal or signal. Set-top input switching method How to switch from TV broadcasting screen to set top (HDMI1) Press the Home or Menu button on the TV remote control to select the input you want to switch from the external input. Check and reconnect HDMI cables when using set-top
    1.We recommend using genuine cables. If the authenticated HDMI cable is not used, the screen may not be visible or a connection error may occur.
    2. Please check the genuine logo. Check the print or imprint of “Ultra High Speed HDMI” on the HDMI cable.
    3. Connect the set-top box and TV to check if external input connection lines such as HDMI are connected normally when watching the broadcast. HDMI Cable Test Method
    1.Click Home or Menu on the TV remote control to select “Settings,” “Customer Support,” “Self Diagnosis,” and “Start Signal Information.”
    2. Please select “HDMI Cable Test” and “Start Test”. Depending on the condition of the cable, it may take 1 to 3 minutes, so please wait.
    3. If the cable is normal, no problem is found.
    4. When diagnosing HDMI cable door zero, check the connection status and replace it with a genuine cable. Set-top power checking method
    1.If there is no set-top power light, please reconnect the power.
    2. When the set-top power is on standby, use the set-top remote control to turn on the power.
    3. If the power is not turned on, please contact the set-top manufacturer.
