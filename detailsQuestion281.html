I can't connect to the set-top box with the TV remote control

  • If the integrated remote control is not set or malfunctions with the TV remote control, there is an obstacle or misset between products. Integrated remote control setting method
    1.First, check if the device you want to set up and the smart remote control are connected to the TV.
    2. Press the Home or Menu button on the TV remote control to select “External Input” and “Integrated Remote Control Settings.”
    3. Please check if the power of the device is on and press the start button. Please select the type of device to set the integrated remote control.
    4. If the set-top box you want to set is directly connected to the TV, please select “Connect directly to the TV.”
    5. If the set-top box you want to set is connected to the TV through the home theater, please select “Connect through the home theater”.
    6. Enter a “residential area” to find a faster service provider and set up your TV. Set up a broadcasting service provider to experience various broadcasts.
    7. Select the external input terminal of the TV to which the set-top box you want to set is connected. Use a simple test to see if you can operate the set-top box.
    8. Select the channel up/channel down button on the screen and check the right screen.
    9. If the channel has changed, select the “Yes” button, and if the channel has not changed, select the “No” button.
    10. Settings have been completed. Now try all the devices connected with one Samsung smart remote control. Precautions for Installing Integrated Remote Control
    1.If there is a blockage due to an obstacle between products when installing the integrated remote control, the reception status may cause a failure in installing the integrated remote control.
    2. Remove obstacles between products and proceed with the installation of integrated remote control.
