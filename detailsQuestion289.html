Please tell me how to set up the 2015 TV automatic channel.

Function guide Automatic channel setting method Check DTV channel selection - Menu → Channel / Broadcasting → Select DTV channel → Select wired General: Select when watching digital broadcasts by installing a separate digital antenna Wired: Select when watching digital broadcasting using wired/cable broadcasting (In the case of apartments, they are divided according to the broadcasting equipment of the management office) Check the automatic channel setup

  • - Menu → Channel / Broadcasting → Automatic Channel Settings → Select "Start" → Automatic Channel Settings (Do not stop until 100%) → Automatic Channel Settings Completed DTV channel selection
    1.Select "Channel/Broadcast" from the menu
    2. Go to 'Screen Mode'
    3. Select cable broadcasting
    ※ Notes General: Select when watching digital broadcasts by installing a separate digital antenna Wired: Select when watching digital broadcasting using wired/cable broadcasting (In the case of apartments, they are divided according to the broadcasting equipment of the management office) Automatic channel setting
    1.Select Menu → Channel / Broadcasting → "Automatic Channel Settings"
    2. Choose Start.
    3. Start 'Automatic Channel Settings' Please wait until it is 100%. If you stop, you can not set up an automatic channel. )
    4. Automatic channel setup complete.
