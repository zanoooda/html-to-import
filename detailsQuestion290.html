Please let me know how to make a 2015 TV preference list

Function guide How to create a preference list ■ Preference list - The channel set as the preferred channel is displayed ♥ on the channel editing or channel list screen - You can create up to 5 preference lists, so you can create a preference list that suits your taste ■ Path - Menu/123 → Menu → Channel / Broadcasting → Edit preferred list ■ Meaning of display icon in channel list : Analog channel : Channel set as preferred list : Channel with channel lock ■ Use channel list
1

  • Select 'Channel/Broadcast' from the menu
    2. Go to Edit Preference List
    3. Select all channels you want to register
    4. Select a preference list (up to 5 can be created)
    5. If you select Registration, ♥ is displayed
    6. Options → Rename preferred list
    7. Select a character from the keyboard using a pointer
    8. Name change completed
