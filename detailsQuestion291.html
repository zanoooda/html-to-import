Usage guide Time setting method What time setting? - You can set the current time automatically or manually - After setting, press the menu/123 button of Smart Control and check in “Information Display” “Current time setting” path (Menu/123, menu includes default path) - System > Time Setting > Current Time > Time Setting Mode Operates when receiving antenna cable connection and digital broadcast signal Connect the cable receiver or satellite receiver to the HDMI terminal or component terminal to manually set the current time when watching TV If the current time set automatically is different from the actual one - System > Time Setting > Current Time > Time Offset It can be adjusted up to -12 hours to +12 hours at 1 hour intervals Only when the time setting mode is set through automatic / network Enter the current time directly - System > Time Setting > Current Time > Current Time Setting Time setting mode allows you to enter the date and time of the manual/current time Use channel list
