UN55H6900AF, please let me know the simultaneous screen function

Question content I am using the UN55H6900AF model, is external input + external input impossible among simultaneous screen functions? For example, I want to connect two PCs and use them as simultaneous screens

  • Answers The simultaneous screen function does not support external input + external input signal, and the input signal on the main screen is only available when connecting components or HDMI On the sub-screen, only digital channels using RF (antenna) can be viewed .
