  • Please refer to the operation sound release method below and the voice guide off method. Operation sound release method
    1.Press the home key on the remote control to go to Settings.
    2. Select Go to Sound on the screen and then go to Expert Settings.
    3. Select Move to Motion Sound.
    4. Select Move to Motion Sound and then Move to Off. How to turn off voice guidance
    1.Press the home key on the remote control and move to select Settings.
    2. Select Go to General on the screen.
    3. Select Go to Accessibility.
    4. Select [Voice Guide Settings].
    5. When you select Voice Guide, it changes to Off.
