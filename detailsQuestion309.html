Please let me know how to register, delete, and edit TV channels in 2017.

Function guide
※ Target model: 2017 smart TV model The 2017 TV channel lock can be set by entering the pin number after checking the channel lock application on the channel / broadcast, and please refer to the method below

  • METHOD FOR SETTING AND RELATING CHANNEL LOCK
    1.Press the home key on the remote control and go to LiveTV to select Channel List.
    2. Select Go to Edit on the full channel.
    3. Full channel edit screen.
    4. Select the channel you want to lock the channel and then go to [Channel Lock Settings].
    5. Enter the PIN number in the channel lock settings. - The initial PIN number is 0000.
    6. This screen is set to channel lock.
    7. To unlock a channel, select a locked channel and select Unlock Channel.
    8. Entering the PIN number unlocks the channel. How to add channel preference list
    1.Press the remote control's home key and go to Live TV to select Channel List.
    2. Select Go to the Preferences list on the full channel.
    3. Select a list to add preferred channels.
    4. Edit Preferences list screen.
    5. On the Select Channels screen, select Add Channels.
    6. On the Add Channel screen, select End X.
    7. This is the screen with the added channel. - On the channel added to the preference list ♥ (heart shape) Can be checked. METHOD FOR DELETING AND RECOVERING CHANNEL
    1.Press the remote control's home key and go to Live TV to select Channel List.
    2. Select Go to Edit on the full channel.
    3. Select the channel you want to delete from the full channel edit and then go to Delete.
    4. Or select a deleted channel and then go to Restore Channels.
