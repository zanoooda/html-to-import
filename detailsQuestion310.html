  • Guided below Choose from two methods So please turn off the store mode. Setting method-1
    1.Press the home key on the remote control and move to the left to select Settings.
    2. Select Go to General on the screen.
    3. Select Go to System Management.
    4. Select Go to Enabled Mode.
    5. Select Go from Store Mode to Home Mode. Setting method-2
    1.Click the home key on the remote control and select Go from Settings to Store Mode.
    2. On the Easy Menu screen, select Go to Usage Mode.
    3. Select Go from Enabled mode to Home mode.
