2018 Q series, home screen does not disappear.

Function guide Corresponding model - 2018 Q-series / UHD TV as a whole (NU7100 or more) Symptom - Home screen (EDEN Menu) does not disappear Method of action - Set 「Automatically Run Smart Hub」 to OFF

  • → Home – Settings – General – Smart Functions – Smart Hub Auto Run 「OFF」 Go to Home - Settings Go to [General] Go to [Smart Function] Change to Smart Hub Auto Run [OFF]
