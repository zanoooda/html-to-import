Please let me know about the 2018 TV game mode setting function.

Function guide
※ Target model: NU8000 or higher model released in 2018 However, 49NU8000 and 49Q6F models are not supported

  • Game mode setting detailed function guide Game mode settings are provided in '1 Game Mode, 2 Game Motion Plus, 3 FreeSync' The main functions and activation methods are as follows. [Note] The screen may blink when switching the function settings. 1 .Game mode Implement optimized image quality and response speed when connecting game consoles. Automatic switching must be supported by the game console specifications.
    ※ Game console means video game console such as PlayStationTM or XboxTM When using game mode, screen mode and sound mode are automatically set to game. 1) When the TV is turned on, the home button of the TV remote control ( Press ). 2) Choose Settings - Menu. 3) Select Menu – General. 4) General – Select external device management. 5) External Device Management – Select Game Mode Settings. 6) The default value of game mode is turned off. 7) Select Game Mode Off as On. 2 .Game Motion Plus It is a function for implementing moving videos. - Displays smoothly and clearly in RPG, adventure games, etc. 1) When the TV is turned on, the home button of the TV remote control ( Press ). 2) Choose Settings - Menu. 3) Select Menu – General. 4) General – Select external device management. 5) External Device Management – Select Game Mode Settings.
    6. Select Game Motion Plus Settings.
    7. Game Motion Plus Settings – Screen blur removal, screen tremor removal can be set from 0 to
    10. 3 .FreeSync It is a function that compensates for screen drag, break, and disconnection in the game. It only works on content that supports Radeon FreeSync. The resolution of the external device is recommended to be 1080p @ 120Hz. You can check the resolution that can be supported for the FreeSync function. FreeSync mode setting: Off / Advanced (48-120Hz) / Basic (90-120Hz)
    ※ FreeSync mode supported by external devices may be different.
    ① Resolution (dotx line): 1920 x 1080, 2 display format: 120 Hz, 3 horizontal frequency (KHz): 135,000
    ④ Vertical frequency (Hz): 120,000, 5 clock frequency (MHz): 297,000, 6 Scan: progressive
    ※ This function may not be supported depending on the model.
    1.When the TV is turned on, the home button of the TV remote control ( Press ).
    2. Choose Settings - Menu.
    3. Select Menu – General.
    4. General – Select external device management.
    5. External Device Management – Select Game Mode Settings.
    6. FreeSync default is off.
    7. For FreeSync, please set it to [Basic] or [Advanced]. Basic: 90 ~ 120Hz, Advanced: 48 ~ 120Hz
