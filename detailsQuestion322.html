Usage guide
※ Target model: 2015 TV model To record a broadcast, you can record or delete the program you are currently watching, and you can book a recording of the program that is scheduled to air

  • Preparations before recording the broadcast; To schedule a recording, you must first set the current time. USB devices for recording are recommended for USB HDDs over 5,400 rpm.
    ※ Device that has completed device performance inspection Digital channel via RF (antenna) cable
    ※ Please note before using User Manual / e-Manual recording and recording reservation. Broadcast recording
    1.Press the [Menu/123] button on the remote control.
    2. Under the virtual remote control ● Select a button.
    ※ Select the program that is currently on air and press the Enter button for a long time. You can choose to record the recorder right away and record it right away. Also, select a program that is scheduled to air and press the Enter button for a long time. You can schedule a recording by selecting the recording reservation that you chose to view in detail. Deleting a Recorded Broadcast
    1.Press the [External Input] button on the remote control.
    2. Select a USB device.
    3. Settings in the upper right corner ( ) → Select 'Filter: Recorded Content'.
    4. Select Settings → [Delete] → Content in the upper right corner.
    5. Select [OK] in the Delete or Not window. Booking a broadcast
    1.Press the [Menu/123] button on the remote control.
    2. Select Broadcast Guide.
    3. Select the program you want to book.
    4. Press and hold the Select button on the remote control and select Reservation Management.
    5. If you do not have a reservation program, select Reservation List.
    6. Select [Recording Reservation].
    7. Set the date and time to book.
    8. Your reservation is complete.
