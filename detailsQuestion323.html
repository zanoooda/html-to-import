    ⑤ (Smartphone) Guide on how to set up magic screen on TV
    1.On the TV screen, Press the (Magic Screen) button.
    2. After entering Magic Screen mode, you can set 'Information, Deco, Photo, Settings, Background Screen' at the bottom.
    ※ You can select other contents by pressing the button before the remote control. 1) Information – You can choose between Basic, Headline News, and Weather 2) Deco – Selectable from mountains, bead clocks, water, and Vrlec pallets 3) Photo – You can choose between board, mono, and duo 4) If you select Settings, you can make changes such as brightness, color tone, automatic brightness adjustment, and ambient off reservation. Settings - You can set '-10 ~ +10' when selecting brightness. Settings – When selecting color tone, you can set 'Cold 25 ~ Warm 25'. Settings – ‘On/Off’ can be set when selecting automatic brightness adjustment. Settings - Ambient (Magic Screen) off reservation can be set to 1 to 4 hours. 5) The wallpaper setting is at the bottom right After selecting the icon, select the wallpaper content to change.
