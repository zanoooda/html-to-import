Function guide Gallery app? Samsung Cloud Service, an application that allows you to synchronize and view shared photos and videos


  • ※ TV support target: Q-series, NU7100 or higher model Start Eden: Q-series, NU7100 or higher support Voice support: Q-series, NU7400 or higher support Terms of use; Supports services that link smartphones, TVs, and Gallery apps It is based on cloud services and must be set to turn on synchronization.
    ※ Mobile cloud synchronization shipment status is 'off', so please set it to 'on'. You must log in with your Samsung account. (The basic condition for interworking between periods is Samsung account) The TV can only view, and deletion and setting are possible on the smartphone. Setting method Smartphones and TVs can run the app after logging in to Samsung account. Select Gallery app on TV Eden → Log in to Samsung account → Enter Pin code (initial setting once) → Check content → Select content using remote control direction key.
