USB ODD connection guide When connecting a USB ODD device, a spec must be compatible between the TV and the ODD device for use

  • The connection can be used by connecting the USB terminal on the back of the TV and the USB terminal on the ODD device, and only models with AV connectivity function among the ODD product specifications can be used by connecting to the TV. Required Spec Description Requisite spec A detailed description ODD interface A/V connectivity (connectivity) function must be supported. TV cannot be connected if A / V connectivity is not supported on ODD device. Connection Only TV USB (HDD) terminals can be used - 5 V 1.0 A spec is used for USB (HDD) terminal - Check voltage / power spec when purchasing ODD device Encoding The codec of the ODD device must match the codec supported by the TV ■ Note What is the [AV Connectivity] function? Ordinary ODD cannot be used by connecting to a TV, but this product refers to a function that allows USB flash memory such as a TV to be used like USB memory on available devices. ■ How to switch to AV connectivity mode
    1.If you touch the open/close button twice within 1.5 seconds while the ODD tray is open, it switches to AV system support mode and the LED remains lit.
    2. After that, whenever the open/close button is touched twice, ODD is repeatedly switched to normal ODD mode and AV system support mode.
