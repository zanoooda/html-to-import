The connected STB power does not work when the TV power is operated.

Expected cause It may occur because power synchronization is not set

  • Method of action Confirm integrated remote control setting It may occur because the integrated remote control is not set. Please use it after setting the integrated remote control. IR extended remote control connection It may occur because the IR extension cable is not connected or the location of the extension cable is wrong. Check the connection of the IR output terminal on the back of the TV Check the location of the IR extension cable Power Synchronization Settings
    1.Enter the integrated remote control setting by pressing the external input
    2. Select the device set next to Start
    3. Select TV Power Synchronization
    4. Check after displaying synchronization V check of the connecting device HDMI-CEC Off Settings If the STB power does not work at the same time when the TV power is running, you must change the Anynet+ setting to Off. Please change Anynet+ setting to turn off and use it. Menu → System → Anynet + → [Turn off -> Change on] HDMI-CEC Off Settings Menu - System Anynet+ Turn off → Change on
