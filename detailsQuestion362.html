Smart remote control cursive function cannot be recognized.

Expected cause It may occur while watching OnTV or because the menu is running

  • Method of action Check Live TV selection The cursive recognition function is not available when watching OnTV other than Live TV status. [Method of Action] Select Live TV or press the Smart Hub key to set TV viewing on the full screen Close the menu window and check the use If the menu setting window is activated, the cursive recognition function is not available. [Method of Action] Please close the menu by pressing the return button of the remote control for 1 second.
