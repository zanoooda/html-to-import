There is a stop or error when running the application.

Expected cause It can be caused by network status, product firmware, and smart hub version

  • Method of action Check network connection
    1.Menu - Network - Network Status
    2. Check if the network connection is normal.
    ※ If both router and globe are marked [X], select IP setting to check if IP setting method and DNS setting method are automatic.
    ※ If the normal globe is marked [X], contact the Internet service provider, call the MAC address on the right, and get a Mac authentication. Check the latest version TV software upgrade (the latest version may vary from update method to update method)
    1.2011-12: Menu - Customer Support - Software Update - Select an update that fits the item.
    2. 2013-14: Menu - Customer Support - Software Update - Select Update Now. Keep Smart Hub up-to-date
    1.2011 ~ 12 years: Update phrase appears when running smart hub with network normal connection and update is performed when selecting update.
    2. 2013: Smart Hub - App Panel - View more apps - Update if there is an update item at the top.
    3. 2014: Update when entering the app panel with the Internet connected. Connect to a hotspot on your smartphone Smartphone hotspot setting method
    1.Menu - Run Preferences
    2. Run additional settings
    3. Tethering and running portable hotspots
    4. After selecting to turn on the portable WI-FI hotspot Select Confirm Warnings Connect hotspots through TV network settings
    1.Menu - Network
    2. Network configuration
    3. Select a hotspot to connect to and enter a security key
    4. Connection complete. Application reinstallation
    1.2011: Smart Hub - [C button] - Select the application, delete it, and reinstall it in Samsung Apps 2 .20 12 years: Smart Hub - Move to the application - [Simple Menu] - Reinstall
    3. 2013: Smart Hub - Apps Panel - More View Apps - Top Options - Select Delete within Options - Select Apps - Select Delete Items at the Top - Reinstall from Sam Apps
    4. 2014: Smart Hub - Apps Panel - Press the Select button for 3 seconds in the item to delete - Delete - Find and reinstall the application in the category Smart Hub Initialization
    1.2011: Smart Hub - [D Button] - Initialization
    2. 2012: Smart Hub - Simple Menu - Settings - Initialization
    3. 2013: Menu - Smart Function - Smart Hub Initialization
