Please let me know how to set HDMI UHD Color (HDMI 2.0).

Mode change method - Menu → Screen → Add-ons → Select "On" in HDMI UHD Color

  • (4K (4:4:4) Changed to 60P support) Support model - UHD support model after 2014 Whether to support HDMI
    2.0 If you change HDMI to turn on HDMI in HDMI UHD Color in screen mode, it supports 4:4:4 (4K 60P) supported by HDMI
    2.0. Setting HDMID UH color
    1.Press the remote control menu button to select the screen.
    2. Moving to Additional Settings
    3. Select HDMI UHD Color.
    4. Select the terminal you want to change.
    ※ However, MHL terminal is excluded ("HDMI 3 (MHL)" or "HDMI 4 (MHL)" terminal disabled)
    5. If you want to change to 4:4:4, "on", If you want to keep existing, choose "off".
    6. Each of the On and Off settings must be executed.
    ※ Automatic power off / on
    7. HDMI 1 to 4 can be supported
    ※ However, MHL terminal is excluded ("HDMI 3 (MHL)" or "HDMI 4 (MHL)" terminal disabled - HDMI
    2.0 cannot be converted)
    ※ Notes ○ When setting on / off, you must run HDMI 1 ~ 4 (excluding MHL terminal). (1-4 Cable batch setting is not possible) ○ Power is automatically turned off/on every time it is set.
