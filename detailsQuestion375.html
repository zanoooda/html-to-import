The screen is erased to one side when connecting to the PC.

Expected cause It can occur with resolution settings and graphics card settings

  • Method of action Optimal resolution setting Please refer to the TV manual to check the optimal resolution of the TV and use it.
    1.Check the optimal resolution among the PC input parts of the TV manual
    2. Select [Screen Resolution] from PC Desktop
    3. Select the optimal resolution of the TV manual and change it TV screen size setting You need to select the TV screen size setting as the original size so that you can watch the normal PC screen without any symptoms of the screen being cut off. Please change the screen size to the original size and use it.
    1.Menu - Select Screen
    2. Screen adjustment selection
    3. Screen size selection
    4. Source size selection Confirm graphic card settings The method of change varies depending on the graphic driver version and manufacturer, so please check the PC graphic card.
    ※ Intel graphics
    ① Right-click Graphic Properties
    ② Click Advanced Mode
    ③ Display - General Settings - Click Customize Street / Sero Ratio to adjust the ratio to 100%
    ※ Nvidia graphic
    ① Open nVidia Control Panel
    ② Display item selection
    ③ Select Desktop Size and Positioning
    ④ Adjust landscape/vertical to screen size
    ※ AMD graphic
    ① AMD Graphics
    ② Right-click Desktop Open Catalyst Control Center (CCC)
    ③ Graphics à Desktop & Display Selection
    ④ Select the lower monitor icon (click the mouse right button)
    ⑤ Select Configuration à Select Expansion Options tab
    ⑥ Overscan 0% adjustment If it has not been resolved, I recommend you to check it through an accurate diagnosis by the engineer.
