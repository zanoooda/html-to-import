I can't see the subtitles when playing the video.

Expected cause It may occur because the properties of the subtitle file are changed or the subtitles are not set

  • Method of action Check the title of the video file and subtitle file The name of the video file and subtitle file must be the same. If the titles are different, please modify the title of the file to be the same. . You can only modify the title of the video file and subtitle file on the computer. How to turn on the subtitle setting Symptoms that subtitles do not appear when playing videos may occur because subtitles are not set. You can change the setting method to turn on the subtitle setting after playing the video.
    ※ How to turn on subtitle settings During video playback → Simple menu of remote control → Subtitle setting → Subtitle → On. Play video → Simple menu Turn off subtitles → Select On Check with another subtitle file This may occur if the properties of the subtitle file are changed. When changing the properties of a subtitle file, the PC displays normal subtitles, but in the case of a monitor, subtitles cannot be displayed if the properties of the subtitles are changed.
    ※ How to take action
    1.You have to download the subtitle file to another file and check it.
    2. Using the encoding program, video files and subtitle files can be combined into one to display normal subtitles when encoding into new files.
