Expected cause It may occur due to malfunction or communication with the set-top box

  • Method of action Initialization check TV malfunction symptoms may cause off / on symptoms while watching. In this case, please reset it to factory shipment status and use it.
    ※ caution : Initialization must be carried out after initialization. Press the remote control home button Move [Settings] at the bottom left Menu → Move customer support Select [Self Diagnosis] Select [Initialization] Password [0000] Enter Select [Initialization] [Check] Reboot when selected and initialization completed Check HDMI cable standard and replace cable It is recommended to use a genuine cable when connecting the TV and the set-top box with an HDMI cable. If the certified HDMI cable is not used, the symptoms of turning off during viewing may occur repeatedly due to communication errors. The genuine HDMI cable is printed or imprinted on the cable with Ultral High Speed HDMI, and if there is no corresponding phrase Please replace it with a genuine HDMI cable with an uncertified cable and use it. Check the "Ultral High Speed HDMI" print or imprint on the HDMI cable. Power cord, reconnect HDMI cable TV malfunction symptoms may cause off / on symptoms while watching. In this case, it is necessary to use after reconnecting the power cord and HDMI cable of the TV. Reconnect after disconnecting HDMI cable Reconnect after disconnecting power cord Check event log In the case of 2019 products, the event log function is applied, and a function to check the operation of the TV is added due to external devices. Whether it is turned off and on due to TV abnormalities using the event log function, or the symptoms of turning off and on due to abnormal signals on the set-top To see if it's happening. [Setting Method] Home → Menu → Customer Support → Customer Support Information → Under Direction Key → Check Event Log Event log information No event log information If there is information in the event log, the TV is operated by the set-top box, and if it matches the day of intermittent off symptoms You need to check the set-top box. If there is no information in the event log, it is necessary to update the TV to the latest version and use it if it is not malfunctioning by an external device. Check HDMI-CEC connection setting If the TV and the set-top box are connected to the HDMI terminal, intermittent power off symptoms may occur due to HDMI-CEC communication error. This is not a product problem, but a symptom caused by incorrect communication between the TV and the set-top box, and can be set so that communication through HDMI is not possible. Menu → System Anynet+ (HDMI-CEC) When checking to turn on Anynet+ (HDMI-CEC) Select Anynet+ (HDMI-CEC) off
    ※ reference HDMI-CEC is not a function applied only to Samsung products, but a communication standard used worldwide Misoperation may occur due to different communication contents used by Samsung and communication contents used by set-top box manufacturers, Even if you choose to turn off HDMI-CEC, you can use the same video / voice transmission through HDMI terminal.
