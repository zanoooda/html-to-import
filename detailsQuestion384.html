STB (set-top box) color abnormalities, red screen, and black and white screen symptoms appear.

Expected cause The cable connection may be incorrect or may be caused by a set-top box abnormality

  • Black-and-white Redness/color abnormalities Color blur / overlap / double Method of action Broadcasting signal check guide If the color appears to spread only on a specific channel, it may occur as a broadcast signal abnormality, not a product abnormality. This is because if there is an abnormality in the product, the same color spreads on all channels. In this case, please contact the broadcaster to check the broadcast signal [wired, cable, skylife, IPTV, etc.]. Self-diagnosis method If the color looks smudged, it can be caused by a broadcast signal problem, and the image quality test function can check whether the product is abnormal or not. How to check image quality test Select [Menu → Customer Support → Self Diagnosis → Image Quality Test].
    1.Press the home button of the remote control
    2. Move lower left [Settings]
    3. Menu → Move customer support
    4. Select [Self Diagnosis]
    5. Select [Start Screen Test]
    6. Check the sample image shown on the TV
