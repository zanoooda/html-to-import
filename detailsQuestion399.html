TV PLUS (TV Plus) network status is not smooth.

Expected cause The screen may be disconnected or stopped due to network speed

  • Method of action Network checking method
    1.The recommended network connection speed is 10Mbps.
    2. It is necessary to check whether the wired or wireless network is connected normally.
    3. You need to check after connecting the TV and the network using mobile phone (mobile) tethering.
    ※ For hotspot (mobile) tethering settings, please contact the mobile phone manufacturer. Network setup method TV products before 2017 : Remote control menu → network → network setting → network name selection → connection TV products since 2017 : Remote control home button pressed → Settings → General → Network → Network Settings Execution → Select Wireless → Select the wireless router or mobile phone (mobile) tethering device you are using → Enter the wireless router password → Complete the connection
    1.Remote control Home button Please press .
    2. Set Please choose.
    3. General Please choose.
    4. network Please choose.
    5. Execute network settings Please choose.
    6. wireless Please choose.
    7. The wireless router you are using Please select my cell phone (mobile) tethering device.
    8. Wireless router (Mobile phone tethering device) Enter a password. Do it, please.
    9. I'm connected to the Internet.
    ※ Network name and password can be found on the setting screen of the wireless router, and please contact the router manufacturer for the wireless router setting screen. If you can't connect by the above method, please get an accurate diagnosis through our professional engineer.
