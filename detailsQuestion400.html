Please let me know how to initialize among the 2014 TV self-diagnosis functions

Initialization method [Click 'More buttons' on the remote control] → [Menu] → [Customer Support] → [Self-diagnosis] → [Initialization] However, if it is not a smart remote control, please press the menu button


  • ※ Delete additional installed apps and change the screen and sound adjusted by the user to the initial value. [Click More Buttons] [Click the menu button on the screen] [Customer Support Press] [Self-diagnosis Press] [Initialization Press] [Click the OK button (center of direction key)] [Number "0000" if password is not set] ["Yes" press (TV reboot)]
