I can't get a voice after connecting the home theater / audio.

Expected cause If you select to turn off the audio return channel among the functions of the home theater, there is no voice on the TV

  • Voice may not appear on the DVD or CD being played, depending on whether it is a scratch or a fixture. Method of action Check the automatic setting of the audio return channel Among the functions of the home theater menu, please automatically change the audio return channel from off. Check the automatic setting of the audio return channel Menu → Select "Settings" Settings → Select "Sound" Sound → Select "Audio Return Channel" Select Audio Return Channel → "Automatic" Title damage confirmation If the title being played (BD, DVD, CD, etc.) is damaged or there is a scratch/foreign object on the surface, voice may be cut off or squeaky. If it's a foreign object, check it after washing it. If there is a scratch or foreign object as shown in the figure, normal output is not possible Check whether it is genuine or not If the title (BD, DVD, CD, etc.) is a supply or downloaded CD, the sound quality may deteriorate.
