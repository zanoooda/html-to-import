Integrated remote control battery consumption is fast. (Sudden discharge)

Expected cause Battery consumption may increase due to integrated remote control usage

  • Method of action Although the same battery as the existing IR remote control is used, the integrated remote control can be used for about 3 months based on AAA battery type due to increased power consumption compared to the existing IR remote control due to voice recognition and touch pad using Bluetooth method. (The usage time may change depending on the actual usage conditions. ) ■ How to save battery consumption If you don't usually use the remote control, put it down without holding it in your hand. → If you hold the remote control in your hand, power consumption occurs in the process of preparing a preliminary touch on the touchpad.
