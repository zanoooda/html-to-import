When you turn on the TV, the smart hub runs at the bottom.

Expected cause The start screen setting is supposed to be turned on when the TV is turned on, so the following screen may occur

  • Method of action How to turn off the start screen setting After connecting to the network on the TV, the initial value is set to run simultaneously with the smart hub on the start screen. Menu → Smart Hub → Start Screen Settings → Auto Start → Select Off / On
    1.Menu → Smart Hub → Start Screen Settings
    2. Auto Start → Select Off
