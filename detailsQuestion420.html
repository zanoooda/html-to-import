What is the function of film mode, please tell me the difference between automatic 1 and automatic 2.

Function guide What's film mode? - You can set the optimal image quality for watching movies

  • When watching a movie, watch it in the mode you want.
    ※ It can only be used when watching TV or when the input signal is external input, component (480i, 1080i), HDMI (1080i). The difference between automatic 1 and automatic 2 in film mode
    1.The movie is shot on 24 Progressive.
    2. The broadcast is conducted at 60 Interface
    3. In order to send a movie, the broadcast goes through a 3:2 process and sets 24 Progressive to 60 Interface.
    4. On TV, we think that the signal from the broadcast is 3:2 when the original signal is sent to the movie, and we consider this when making 60P by de-interlace.
    5. The menu used at this time is film mode. - Automatic 1: ignore the subtitles of the movie and de-interlace. (The caption may be broken) - Automatic 2: De-interlace considering the subtitles of the movie. (The caption does not break) Film mode setting method Choose Menu → Screen → Additional Settings → Film Mode [Off, Auto 1, Auto 2]. Select [Film Mode] Select from [Off / Automatic 1 / Automatic 2]
