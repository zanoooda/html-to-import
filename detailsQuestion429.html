Please let me know how to set up a multi-link screen (2 division support model).

Expected cause Inquiries may occur when using the two-part multi-link screen function

  • Method of action Support model The multi-link screen function supports more than 14 years of H7100, and the support specifications support up to 2 divisions and up to 4 divisions depending on the model and conditions.
    1.Only web browsers, YouTube, and Samsung Apps are available.
    2. The H7100 ~ H8000 model supports up to two divisions, and the HU7000 ~ HU9000 model supports up to four divisions. Two-part multilink screen entry method
    1.How to enter through the menu: Menu - Screen - Run Multilink Screen. Initial entry is basically a web browser and searches for information related to ON TV settings. Menu → Screen Multilink screen run Multilink screen execution screen
    2. How to enter through smart remote control: Run the “Multiscreen” button. Select the multi-screen button at the bottom of the remote control Multilink screen execution screen Multiscreen condition change method
    1.Changeable icons: Web browser, YouTube, Samsung Apps, etc. Running multilink screen Select the multi-screen button Select the desired icon Multilink Screen Operation Specifications
    1.Web browser is viewing screen + Internet
    2. YouTube is viewing screen + YouTube list screen
    3. Samsung Apps View Screen + App Panel Operation Specifications for Each Icon
    1.Web browser (up to 2 divisions of H7100 to H8000 models are supported)
    ① When running a web browser, ON TV searches for related information based on current broadcasting information.
    ② If ON TV is not set, search for popular search words.
    2. YouTube (H7100 ~ H8000 model supports up to 2 divisions)
    ① When you select YouTube, ON TV searches for related videos based on current broadcasting information. 2 If ON TV is not set, search for popular search words.
    3. Samsung Apps (H7100 ~ H8000 model up to 2 divisions support)
    ① When running Samsung Apps, the Apps panel is displayed, and applications that cannot operate during multi-screen functions are deactivated. METHOD FOR SETTING TV SPEAKER IN MULTILINK SCREEN FUNCTION
    1.While running the multi-screen, press the multi-screen button on the remote control - Select Settings. Running multilink screen Select the multi-screen button Select the desired icon
    2. Select Speaker Settings - Select TV Speaker:OO, and then select the screen you want to output the speaker. If selected automatically, the sound of the item with the cursor moved through the remote control is displayed on the split screen. Running multilink screen Select the multi-screen button Select the desired icon
    2. Select Bluetooth headphone connection and press the pairing button on the Bluetooth earphone. When the earphone item to connect appears on the TV, press Select Remote Control. When the pairing and connection icons appear, press the Select button one more time to connect and use it. Running multilink screen Select the multi-screen button Select the desired icon
