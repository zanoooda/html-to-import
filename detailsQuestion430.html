Please let me know about setting / releasing ticker (ticker / application / app) automatic execution.

Function guide It is a function that can be executed and used immediately without running a separate app when the TV product is turned on by automatically setting an application that supports the ticker function

    1.(2011 model) This is how to set and release ticker auto run.
    ① Enter Smart Hub - Select Remote Control [D] button
    ② Service management selection
    ③ Ticker auto run selection
    ④ Turn it off or select the service you want
    2. (2012) How to set up and release ticker auto run.
    ① Enter Smart Hub - Select Remote Control [D] button
    ② Setting selection
    ③ Service management selection
    ④ Select auto-run ticker - turn off (or select desired service)
    4. (2014) This is how to set and release ticker auto run.
    ① Menu - Smart Hub - Run App Settings
