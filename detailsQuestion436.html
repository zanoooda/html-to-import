I'd like to know how to set up an integrated remote control for 2014 TV.

Setting method Support model: 2014 H model integrated remote control support model
※ Please proceed with IR expansion cable reconnection and integrated remote control reset


  • ※ First, you need a smart remote control and IR output cable. Setting Integrated Remote Control
    1.Connect the IR extension cable to the IR output terminal.
    2. Place the extension cable in front of the external device IR receiving unit.
    3. Select the "External Input" button on the remote control.
    4. Select "Setting Integrated Remote Control" in the upper right corner.
    5. Start setting up the integrated remote control.
    6. Positioned in front of the device to connect and control the IR extension cable
    7. Select the device you want to set. E.g.) Set-top box
    8. If the voice of the STB is connected separately to the HST, select the TV if the home theater is not selected (external speaker connection).
    9. Under Broadcast Service Provider Settings, select a region.
    10. Select the area you currently live in.
    11. Select the service provider you are currently watching. 12. Select whether to apply the value set by the integrated remote control to the ON TV. (Basic broadcasting source = recommended for ON TV panel) (Assistant broadcast source = added only to ON TV setting value) (ON TV is not set = not applied to ON TV) 13. Select the input condition to which the STB is associated. 14. Select Channel Up/Down to see if it changes. 15. If the channel is changed, press Yes to change the completion, and if the completion is not changed, select No to change the channel again. 16. When the channel change is completed normally, select Complete.
