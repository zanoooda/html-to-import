While watching TV, the phrase hybrid TV appears...

Support model - All models of smart TV in 2013 and 2014 Hybrid TV? Compared to existing broadcasting information that provides services such as broadcasting information, replay, performer information, SNS, voting, and quiz participation for programs when moving to the channel after installing an application related to the channel This feature is a little more upgraded

  • If you watch the relevant channel with the hybrid function turned on and the relevant application installed, the hybrid function is executed. If you do not want to use this function, You can change the hybrid function to turn off. Please refer to the following two ways to solve the problem. Method I) Menu - Smart Hub (Smart Functions) - App Settings - Hybrid TV - Set to Off. [ Hybrid TV “Off” Method ]
    1.Choose Menu - Smart Hub (Smart Features).
    2. Select App Settings.
    3. Select Hybrid TV.
    4. Change the on to "off". Method II) If method I) does not solve the problem, delete KBS ICON and SBS NOW applications. [How to Delete Hybrid TV Related Applications ]
    1.Go to Smart Hub - App Panel - KBS OHTV or SBS NOW application.
    2. In this application, press and hold the Select Remote Control button and select "Delete".
