Connecting PC remotely to TV in 2021

■ Remotely connect to PC
1

  • If the Windows PC or Mac is powered on and the remote access option is allowed, it can be connected directly to the remote PC.
    2. Set the PC environment to use the PC on TV function.
    ※ Windows connection through RDP protocol is supported in Windows 10 Professional version or higher.
    ※ MAC OS connection through VNC protocol is supported by MAC OS X
    10.5 or higher. [How to Establish Windows Remote Desktop Connection ]
    ① Please enter Control Panel in the search bar of the taskbar or select the "Control Panel" app in the Start menu → Windows system. 2 Select “System and Security” and click Allow Remote Access on the System. 3 In the System Properties pane → Remote Desktop, select “Allow Remote Connection to This Computer” and click the Apply button. [ Mac Virtual Network Computing Setting Method ] 1 Select Menu → System → Configuration and then "Share". Select the checkbox for screen sharing and remote login from the service. 2 Select the checkbox for screen sharing and remote login. * In Mac OS, if the resolution is not changed, the previous screen may remain while connecting to the remote PC.
    3. Press the home button on the TV remote control to select "External Input" → "PC on TV".
    4. Enter the IP address or profile of the PC you want to access. Enter your PC's user name and password, and select a protocol.
    ※ Depending on the network situation, screen or voice disconnection may occur, so it is recommended to use wired network connection.
    ※ When PC enters power saving mode, connection is disconnected and reconnection is not possible. It is recommended to turn off and use the power saving mode of the PC.
