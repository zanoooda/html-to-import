The remote doesn't work.


※ If the remote control does not work, it may occur if the remote control pairing is cut off or the remote control battery is discharged or the battery is consumed


  • 1.Take a picture of the front of the remote control through the camera on your smartphone. * For iPhones, it is easier to check with the front camera.
    2. If you see an infrared signal while pressing the power button on the remote control, the remote control is normal. We need to check if the remote control is paired.
    3. If you do not see an infrared signal, charge the remote control through a solar cell or a USB port at the bottom of the remote control.
    4. For the remote control that uses batteries, replace the batteries. The remote control provided may vary depending on the model you purchased. ■ Remote control pairing method
    1.With the TV on, press the Return button and the Play/Pause button at the same time for more than 3 seconds.
    2. The pairing of the TV and the remote control starts. Please proceed with remote control pairing according to the guidance on the screen ■ TV remote control charging method
    1.There is a solar cell on the back of the TV remote control that can charge the remote control.
    2. When the remote control is not in use, the back of the remote control can be placed upward to charge through natural and indoor light.
    3. If the remote control battery is insufficient and does not work, connect the USB cable to the bottom of the remote control and use it after charging.
    4. When charging through a solar cell, be careful not to have the remote control between the sofa gaps or under the table. ■ Initialize remote control (QA60 model ↑) if it is not resolved even after eco remote control pairing attempt Press the Return button and Select button at the same time for at least 10 seconds.
