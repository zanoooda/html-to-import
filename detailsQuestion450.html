Guide on how to disable melody function
※ Please disable the melody function of the 2013 model (F5200 or less) and select the volume size


  • ① Menu - Settings (Initial Settings)
    ② Melody - Select Off
    1.Menu → Settings → Melody → Select the desired size → End
    2. Melody size selectable: Turn off, small, medium, large
    ※ If the volume is "0" or the "quiet" button is pressed, there is no sound when the TV is turned off.
    ※ The sleep reservation function does not work when the TV is powered off.
