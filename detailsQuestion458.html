I can't get a set-top box/converter connection.

Expected cause It may occur due to switching to external input mode or HDMI cable abnormality

  • Method of action Check external input switching If the TV and the set-top box are connected by an HDMI cable, the input mode of the TV must be changed to the HDMI input mode to which the set-top box is connected. Press the home button on the TV remote control. Please select HDMI mode where the set-top box is connected. How to set up a product after 2017 When watching TV broadcasting through the set-top box, you need to switch to connected input mode such as HDMI / component to use the set-top box. Press the remote control home button Select external input and move to connected external device Check the operation status of the set-top box Check if the set-top box power works normally (check power ON status and select channel). If the power is turned on normally, the current channel display appears on the front liquid crystal. If the front liquid crystal is not displayed, it is necessary to check the reconnection of the power cord of the set-top box, If there is no display on the liquid crystal even when reconnecting the power cord, it is necessary to check the set-top box. Power standby status (only time information is displayed as LED) Normal operation status (channel number is displayed as LED) Check cable connection status It is necessary to check whether the TV and the set-top box are connected normally. Please separate and reconnect the cable connected to the back of the TV and set-top box. Remove and reconnect the HDMI cable on the back of the TV Remove and reconnect the HDMI cable on the back of the set-top Remove and reconnect the TV rear component cable Remove and reconnect the set-top rear component cable HDMI cable confirmation guide It is recommended to use a genuine cable when connecting the TV and the set-top box with an HDMI cable. The genuine HDMI cable is printed or imprinted on the cable with Ultral High Speed HDMI, and if there is no corresponding phrase Uncertified cables may not show the screen or cause symptoms such as poor image quality. In this case, please replace it with a genuine HDMI cable and use it. Check the "Ultral High Speed HDMI" print or imprint on the HDMI cable. Check HDMI cable test A self-diagnosis function that checks for abnormalities in the HDMI cable connected to the TV is built into the TV to check for abnormalities in the TV or HDMI cable That we've never seen before.
