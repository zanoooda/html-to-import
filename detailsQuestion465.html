It can be caused by abnormalities such as internal circuitry, panels, etc. Vertical line Horizontal-line Method of action Broadcast signal check confirmation guide (1) If a vertical line occurs only on a specific channel, it may occur beyond a broadcast signal, not beyond a product. If a product abnormality occurs, a line is formed in the same location of all channels, but if a line is generated by a broadcast signal, This is because the location of the line is different for each channel. In this case, please contact the broadcaster to check the broadcast signal [wired, cable, skylife, IPTV, etc.]. Broadcast signal check confirmation guide (2) When you press the menu button on the remote control, you can check for product abnormalities using the menu window displayed. ☞ If the line appears to be covered by the menu window, it is necessary to check the broadcast signal as a symptom of a line occurring due to a broadcast signal abnormality. In this case, please contact the broadcaster (wired, cable, skylife, IPTV, etc.) to check the broadcast signal. ☞ If the line appears on the menu window as it is, it is a symptom of product abnormality and needs to be checked by the engineer. Self-diagnosis method If the line is visible on the screen, it may occur due to a broadcast signal problem, may occur due to TV abnormalities, and the image quality test function can check whether the product is abnormal or not. How to check image quality test [Menu → Customer Support → Self Diagnosis → Image Quality Test] Menu - Customer Support Self-diagnosis Image quality test Checking for abnormality How to check image quality test in 2017 [Click remote control home button] [Move left lower setting] [Menu → Move Customer Support] [Self-diagnosis Selection] [Select to start image quality test] [Sample image stored on TV is displayed] During the image quality test, the high-definition image built into the TV is displayed on the screen, and if the high-definition image does not have a line and is clearly visible, , it is a symptom caused by a broadcast signal problem, not a product problem. In this case, please contact the broadcaster to check the broadcast signal [wired, cable, skylife, IPTV, etc.]. If there is a line in the high-definition image displayed, it can cause circuit abnormalities and you need to check it out.
