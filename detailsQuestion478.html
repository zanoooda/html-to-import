Channel changes are not supported in current mode.

Expected cause This may occur when using the TV remote control when switching to external input mode or changing the set-top box channel

  • Method of action METHOD FOR CHECKING EXTERNAL INPUT MODE This may occur when the channel button is pressed when the TV is switched to external input mode. Press the external input button at the top of the remote control to switch to TV mode. Press the TV/External Input button Go to external input PC → TV METHOD FOR CHECKING EXTERNAL INPUT MODE OF 16 ~ 21 YEAR TV PRODUCT
    1.Remote control Home button Pressing
    2. External input Please choose.
    3. Please select the device you are currently using from the external input. In case of HDMI Edit Please choose.
    4. Device type editing Select the connected device type in Check Please choose. METHOD FOR ADJUSTING SETTOP BOX REMOTE CONTROL - When watching TV broadcasts using a set-top box, the TV only operates the monitor function, so it is impossible to adjust the channel with the TV remote control. When using set-top box connection, channel change should be used using remote control dedicated to set-top box. Press the set-top box remote control volume + (or ∧) button to adjust the volume significantly. Set-top box remote control TV remote control
