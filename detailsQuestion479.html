When playing a video, an unsupported file is displayed.

Check video file compatibility issue In order to play a video on a TV, the information of the video file, such as file format and codec, must be correct to play it on the TV

  • If only a specific file cannot play a video, it may occur because the codec information of the video file is a file that is not used on TV. It can be seen as a symptom caused by the mismatch between file information and TV, not product abnormalities. To check the video file codec, you can install the video encoding program on your PC and check the information of the video file through the encoding program. We cannot provide the program because there is no encoding program operated by Samsung Electronics.
