There is no power ON / OFF button for 3D glasses...

Power ON/OFF method guide ■ Power method Automatic power ON / OFF type: SSG-3700CR, SSG-3300GR, SSG-3300CR Power button type: SSG-3100GB, S3000GR, M3100GB ■ Automatic power ON operation condition Automatic power ON / OFF can be used only when the following three conditions are met

  • Touch sensor + acceleration sensor + 3D signal In other words, the TV sends a 3D signal, holds 3D glasses (touch sensor), lifts glasses, and wears them on the ear (acceleration sensor) ■ Automatic power off operation condition When releasing 3D mode on TV (about 3 seconds) If there is no movement for 5 minutes
