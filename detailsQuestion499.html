Smart TV Wireless Network Configuration

Wireless network setup guide Connect the wireless dongle to the USB2 terminal on the back of the TV as shown in the picture

  • (Models with LED6500 or higher in 2011 are built-in wireless LAN models and are not connected separately.) METHOD FOR SETTING A TV WIRELESS NETWORK
    1.Remote control Menu Me Home button Please press .
    2. Menu → Network → Select Network → wireless Selection
    3. Network Settings → Network selection Execution
    4. Find and select the router manufacturer you are using
    5. In case of general setting Unsecured connection
    6. In case of security setting Security key Selection
    7. Enter the security key item (the security key is an item set by the Internet router)
    8. Network connection complete.
    9. Run network test
