Please tell me how to set the light effect.

Expected cause Light effects can occur while watching or when set to Always

  • Method of action The light effect function that lights up the Samsung logo at the bottom of the TV is set, so you can see the light on the Samsung logo while watching TV. If you see a light on the Samsung logo while watching, please change the light effect [off, waiting]. Turn-off - The light is always turned off in Samsung logo Standby - When the power is off, the light is turned on in Samsung logo Watching. - When the power is turned on, the light is turned on in Samsung logo All the time. - Samsung logo always lights up Menu - System Menu - System Turn off the light effect, select standby
