I want to know how to record TV.

TV recording method guide Support model 2012 LED ES6350F or higher is available Recording method Immediate recording: Connect the external HDD to the TV USB terminal 1 → Select the red [●] recording button on the remote control → Proceed format with recording device → Select the device to format → Automatically check the performance after format → Select and record the recording length after format completion
※ You can stop recording by pressing the ■ button on the remote control

  • Reservation recording: Run smart hub → manage reservation → select button → reserve recording → select desired channel and condition and check
    ※ To cancel the reservation recording, cancel the reservation in Smart Hub → Reservation Management. In order to record, the device format is essential on the TV, and the data is deleted during the device format, so please back up the necessary data to the PC in advance. Immediate recording Pre-recording There is a difference in the method according to the product model, so please refer to the manual of the product model you are using.
