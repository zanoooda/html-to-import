I want to know how to pair the remote control of the smart TV in 2021.

Smart remote control pairing method guide Check Smart Remote Control Support Model ■ 2021 Smart Remote Control Support Model - QLED rechargeable: QA60 or higher - UHD non-chargeable: UA8000 or higher (UA7000 product IR support) Smart remote control support model Model name Q900A, Q800A, Q700A, QA90, QA80, QA70, QA60, LSA03A(Frame) UA9000 UA8000 UA7000 Remote control Guide for pairing with TV ■ Smart remote control pairing method Head the Samsung Smart Remote Control straight to the TV and press the Return button and the Play/Pause button for more than 3 seconds at the same time

  • ☞ Images, buttons, and functions of Samsung Smart Remote Control may vary depending on the model. ☞ If the remote control is directed to the remote control receiver of the TV, the remote control works more effectively. ■ Initialize remote control (QA60 model ↑) if it is not resolved even after eco remote control pairing attempt Press the Return button and Select button at the same time for at least 10 seconds. ■ Initialize smart remote control if it is not resolved even after other smart remote control pairing attempts Press the return button and the multi-key or additional information button for at least 5 seconds at the same time. Guidance on initialization method due to malfunction If you return the setting to malfunction or shipment conditions, you can restore it to factory shipment conditions through initialization applied to the product. ■ Setting method Press the home button of the remote control → Settings → Menu → General → Initialize → Password 0000 → Reboot and complete initialization Initialization setting method Press the remote control home button Press Settings on the Screen General press Initialization press Password [0000] Input Initialization press All settings are initialized, the TV is automatically turned off and on, and the Initial Settings screen appears. When you turn on the TV, it automatically connects to Samsung Smart Control. If it is not automatically connected, head Samsung Smart Control to the remote control receiver of the TV and press the return button and the play/pause button for more than 3 seconds at the same time.
