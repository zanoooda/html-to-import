'21 model smart remote control integrated setting method
    ① Connect the external device you want to set as an integrated remote control to the TV and turn it on. 2 Press the external input button and select the integrated remote control settings to complete the settings according to the instructions on the screen.
    1.Remote control home key → external input
    2. Select external input → integrated remote control setting
    3. Start setting up integrated remote control
    4. Set-top box (Select external device to connect)
    5. Residential area input
    6. Service provider settings (set-top box)
    7. TV external input setting (connection device)
    8. Set-top box control test (channel up / down)
    9. Select “Yes” if the channel has changed
    10. Setup complete.
