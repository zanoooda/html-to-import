I want to know how to switch to TV or external input.

To watch the video, press the Play button (▶)

  • You can change the external input of the TV such as TV, set-top box, game console, etc. ■ External input switching method (how to switch from TV broadcast screen to set top (HDMI1))
    1.Press the home button on the TV remote control to select the input mode to which the external input → set-top box is connected. HDMI connection terminals may vary by TV model. ■ External input switching method (how to switch from set-top broadcasting screen to TV) Press the home button on the TV remote control to select external input → TV. ■ External input switching method (how to switch from set-top broadcasting screen to game screen)
    1.Press the home button on the TV remote control to select an external input → input mode in which the game console is connected. HDMI connection terminals may vary by TV model.
    ※ If the connected HDMI device does not appear when switching external input, please turn off the TV power, disconnect the HDMI cable, and connect again. After that, please turn on the TV power to check if the HDMI device is recognized.
