When watching with TV antenna, "Input signal is weak or no I see a screen that says,


※ "There is no input signal

  • "When a message appears, it may be caused by an abnormality in a broadcast signal or an external device signal, or an incorrect selection of an external input. Check and reconnect antenna cables
    1.If you are watching TV with an antenna, please completely separate and reconnect the antenna connection line between the TV and the wall terminal. External input switching method
    1.Press the home button on the TV remote control to enter externally → Select TV. Broadcasting signal test method
    1.Press the Home or Menu button on the TV remote control to select Settings → Customer Support → Device Care → Self Diagnosis → Signal Information.
    2. If the antenna signal is 4 or less or the SNR value is 20 or less, it is recommended to check the broadcast signal. Signal information is only available on digital broadcast channels. Automatic channel setting method
    1.Click Home or Menu on the TV remote control to select Settings → Channel / Broadcasting → Automatic Channel Settings.
    2. The number of channels searched may vary depending on the region, broadcaster, and surrounding environment.
    3. If the channel or automatic channel setting is inactive, please change the external input to 'TV broadcasting' because you are currently using an external device (set-top box).
