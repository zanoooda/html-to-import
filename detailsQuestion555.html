When you watch it on the set-top, the TV screen is blurred or the picture quality is bad

  • The blurring of the screen on the TV may occur when the broadcast signal is abnormal, the content, or the resolution of the set-top box is low. Image quality test method
    1.Click the home or menu of the TV remote control to set → Customer support → Device care → Self-diagnosis → Image quality test
    2. Please select the "OK" button on the image quality test screen. The test photo will be displayed soon.
    3. Select "Yes" when the test screen appears normally.
    4. If the diagnosis result is normal, it is likely to be a signal from an external device or a cable abnormality. Please check the external device or the connection status.
    5. If the image quality test screen does not appear normally, select "No". Additional test screen appears.
    6. If you have the same symptoms in the additional sample image, you'd better check the TV. Set-top resolution setting method
    1.Press the “Home” button on the set-top remote control to enter the set-top menu.
    2. Please select “MY” from the top menu.
    3. Go to the bottom and select Settings → Screen / Sound Settings.
    4. Please set it according to the resolution you use.
    5. If you don't know the resolution well, choose “Auto”.
    6. For more information on how to set up, please contact each carrier (settop). If the horizontal line, vertical line, or image quality is poor during the image quality test, it may occur due to internal symptoms of LCD panel or TV. Please get an accurate diagnosis from a professional engineer for that part.
