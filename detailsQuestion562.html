Please tell me how to set the TV magic screen to turn off/on automatically.

[Step 1 ] Turn on automatic switching of TV magic screen
①' SmartThings Run the app

  • - Please select 'SmartThings' app.
    ②' Menu Choose. - Please select 'Menu'.
    ③ Selection - Please select 'SmartThings Settings'.
    ④' Automatic switching of magic screen 'Settings - Please set 'Automatic switching of magic screen' to 'In use'.
    ※ Supported model - 2018 TV: Q6F or higher model - 2019 TV: Q60R or higher / lifestyle model (excluding The Frame) - 2020 TV: UT8000 or higher / Lifestyle model Q series all models - 2021 TV: UA8000 or higher / Lifestyle model Q series all models
    ※ Note, you can use step 2 after step
    1.: ' Automatic switching of magic screen After activation. Magic screen auto-on/off function Please use '. [Step 2 ] Magic screen auto on / off setting
    ①' Registered TV Choose. - Please select the TV from the product card on the app screen.
    ② Selection - Press (More View button).
    ③' Magic Screen Settings Choose. - Please select 'Magic Screen Settings'.
    ④' Automatically turn on/off the magic screen 'Settings - Please set 'Automatically turn on / off magic screen' to 'In use'.
