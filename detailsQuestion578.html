Function guide
※ When selecting consent to smart hub terms and conditions during the initial setting stage, check the required items and options before proceeding

  • Smart hub initialization method Initialize smart hub (including magic screen) and Samsung account. Home → Settings → Customer Support → Device Care → Self Diagnosis → Smart Hub Initialization Set Customer support → device care Self-diagnosis selection Smart Hub Initialization Enter your password. Need to reset smart hub terms and conditions and policies after initialization is completed
    ※ Use when there is a problem with smart hub service in network and TV. Start smart hub connection test Home → Settings → Customer Support → Device Care → Smart Hub Connection Test → Success / Failure Set Customer support → device care Smart hub connection test Start smart hub connection test Success screen Failure screen
    ※ Test may fail due to non-agreement of smart hub terms and conditions and network error. In this case, please turn off the TV and wireless router and check the attempt and network status again.
