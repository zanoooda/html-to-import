2015 TV Initialization Method
※ When selecting consent to smart hub terms and conditions during the initial setting stage, check the required items and options before proceeding

  • TV initial setup method Select the network to connect (cross is displayed in the right direction in the upper right corner of the screen) → Consent to the terms and conditions of using the smart hub → Select the TV reception method. Press the remote control selection button when setting the residential area → Select the cable company used after completing the residential area setting → Set the integrated remote control → Complete the setting
    1.Network setup
    2. Check Smart Hub Terms and Conditions Consent Selection
    3. Select TV reception method (set-top box)
    4. Press the Select Remote Control button
    5. Areas are selected in the order of 00:00 – 00.
    6. Select the cable company you are using
    7. Integrated Remote Control Settings – Select Confirmation
    8. Setup Complete – Select Complete
