I can't get a wired network.

Expected cause It may be caused by a change in router connection and settings

  • Method of action Check the connection of the LAN cable It may occur because the LAN cable is not inserted to the end. Please reinsert the TV and the shared climate surface LAN cable before using it.
    1.Please reconnect router LAN line
    2. Please reconnect the TV LAN line Confirm router and IP settings It may be caused by a router malfunction or a change in the IP setting of the TV. Please use it after checking router initialization and TV IP settings again. ■ Reset router power Please power off the router and turn it back on. ■ Check TV IP setting
    1.Run the TV's Menu - Network - Network Status, then select IP Settings.
    2. Select “Accept Automatic Address” for IP setup method and DNS setup method. Confirm router WAN port If the TV and router are marked as normal (V) and the router and the Earth are marked as X, please check the Internet (WAN) connection. ■ Check Internet (WAN) connection The WAN port flashes when the Internet is connected normally. Please use it after checking the external Internet line reconnection in the WAN port on the back of the router. Please install the LAN wire to make a click sound and make sure that the light is blinking near the port. Checking MAC Certification ■ Check authentication for carrier Mac (MAC) address Menu-Network-Execute network status. Please contact the carrier to which the customer subscribes to the MAC address to check for MAC authentication. If it has not been resolved, I recommend you to check it through an accurate diagnosis by the engineer.
