USB / external HDD connection is not recognized.

Expected cause It may occur because the USB memory device is not connected normally

  • Method of action Check USB terminal
    ① When connecting USB memory You can connect to all USB terminals applied to the TV. However, if the USB memory is formatted in the NTFS method, it can only be recognized at the USB (HDD) terminal.
    ② When connecting USB external HDD It can only be recognized at USB (HDD) terminal. If it is connected to a terminal other than the USB (HDD) terminal, it is not recognized.
    ③ When connecting memory or HDD to PC It may not be recognized as memory or HDD or higher. Memory formatting method It may not be recognized as a USB memory file system abnormality.
    ※ Please back up the files stored in USB memory to your computer and format them into USB memory. USB terminal connection When connecting USB external HDD, it can only be recognized at USB (HDD) terminal. It does not recognize when connecting to other USB terminals. [Note] TV USB terminal support specification General USB terminal: File system (FAT), voltage/current (5V, 0.5A) support USB external HDD: File system (FAT/NTFS), voltage/current (5V, 1.0A) support
