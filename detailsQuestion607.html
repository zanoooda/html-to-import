Voice recognition isn't working.

Expected cause It may occur due to network connection failure or malfunction

  • Method of action Check how to use voice recognition - To use the voice recognition function, press the microphone button at the top of the TV remote control to fire the voice command you want to execute. - It can be operated only when the microphone button is pressed during ignition, and if the microphone button is pressed short only once or the button is turned off during voice recognition ignition, normal recognition may not be possible. Check network connection status - If the voice command you want to execute is not displayed on the screen when you press the microphone button at the top of the TV remote control, you should check the network connection status. - If the network is not connected, the voice recognition function is not available.
