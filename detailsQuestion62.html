The present time is not right.

Expected cause It may occur when manually selecting the current time setting It may occur according to the broadcast signal Method of action This is a problem caused by not setting the current time


  • ※ How to set the current time automatically / manually [Menu → System → Time Setting → Current Time → Time Setting Mode (Automatic / Manual)] → It is automatically recommended when selecting time setting mode (automatic / manual), and automatically set through broadcast signal when automatically selected. However, the time cannot be set when automatically selecting when watching external input. It is a symptom caused by no broadcast signal (DTV, ATV) or signal instability, and it is necessary to check whether the broadcast signal is in good condition. When watching external input, the signal is supplied through the set-top box, so the current time may not be right.
