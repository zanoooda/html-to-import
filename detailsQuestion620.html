Please tell me how to delete the application.

Method of action
※ For applications that do not have deletion items, it is an application that cannot be deleted as a basic application

  • Application deletion method
    1.Run Smart Hub → Move the application to delete.
    2. Click the simple menu on the remote control In the pop-up window, select "Delete".
    3. When the pop-up comes up, Select "Yes".
    4. "Deleting..." in the upper right-hand corner. It will be deleted after pop-up
