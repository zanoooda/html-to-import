Screen mode automatically changes to a clear screen.

Expected cause It may be caused by selecting a POP function to display the function of the product

  • Method of action Advertisement window release method In store mode, the ability to display the main features of the product on the TV screen can be applied to display advertising phrases. By changing the store mode to home mode, normal viewing is possible without displaying advertising phrases. ■ Pre-2008 products (excluding Brown tube TV and some models) Menu → System or Settings → Initial Settings → Change Store Mode to Home Mode → Skip → Skip → Check. Press the remote control menu button Press Settings on the Screen Initialization press Press OK on the automatic channel Check in 'home mode' Press the 'Menu' button on the remote control to skip Press the 'Menu' button on the remote control to skip Click check the remote control to complete. ■ 2009 ~ 2012 products Change Menu → System or Settings → Initial Settings → Store Mode to Home Mode. Menu - System Initial setup Store mode → Change home mode Menu Language - Korean - Next Network Settings - Skip Home mode selection Select the Current Time - Remote Control [Exit] button ■ Products from 2013 to 2014 Menu → Customer Support → Usage Mode (bottom of page) → Change Home Mode.
