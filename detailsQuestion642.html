I want to know about digital antenna (UHF) connection and installation.

Expected cause An inquiry may occur when connecting to a TV using a UHF antenna

  • Method of action Antenna installation guide Connect the cable to the cable connection terminal of the UHF antenna and the antenna input terminal of the TV. Connecting Cables to Indoor UHF Antennas Connect cable to TV antenna input terminal Select Menu → "Channel" from Remote Control Channel → Select "Automatic Channel Settings" and click Start to proceed with automatic setting [Before 2006] Menu → Channel → DTV Channel Selection → Wire Selection Select antenna connection status, select the following items when using UHF antenna [As of 2008 ~ 2010] Menu → Channel → Automatic Channel Setting
    1.Menu → Channel → DTV Channel Selection → Wire Selection
    2. Auto channel setup execution [2011 ~ 2014 standard] Menu → Channel → Automatic channel setting Menu → Channel DTV channel selection (general/wired) Auto channel setup execution [2015 standard] Menu → Channel → Automatic channel setting
    1.Menu → Channel
    2. DTV channel selection (general/wired)
    3. Auto channel setup execution [Note] Select DTV channel
    ① General: Install a separate digital antenna to select a digital broadcast when watching.
    ② Wired: Select to watch digital broadcasting using wired/cable broadcasting. In the case of apartments, they are divided into general/wired according to the broadcasting equipment of the apartment management office. → When setting up an automatic channel, set it to DTV channel selection wired and then set up an automatic channel. If DTV broadcast is not received, set DTV channel selection to normal and proceed with automatic channel.
    ※ About 90% of domestic TV broadcasting households use wired / cable.
