How to check image quality test Select [Menu → Customer Support → Self Diagnosis → Image Quality Test]. Menu - Customer Support Self-diagnosis Image quality test Checking for abnormality During the image quality test, the high-definition image built into the TV is displayed on the screen. If the color does not spread to the high-definition image and can be clearly seen, , It is a symptom caused by a broadcast signal problem, not a product problem. In this case, please contact the broadcaster to check the broadcast signal [wired, cable, skylife, IPTV, etc.]. If there is a line in the high-definition image displayed, it can cause circuit abnormalities and you need to check it out.
