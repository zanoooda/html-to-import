I can't hear the TV using the multi-room. (Based on Smart TV 2014 model)

Expected cause It may occur because it is not connected to the network or the speaker is not added

  • Method of action Check TV speaker settings TV remote control menu → sound → speaker setting → multi-room link selection Sound from TV remote control menu Speaker setting Change the multi-room link to the TV speaker Select Multiroom Link Soundbar and Speaker Surround Settings TV remote control menu → sound → speaker setting → setting → multi-room → sound + surround → central sound bar model selection → rear left speaker selection → rear right speaker selection → check
    ※ It is necessary to distinguish when selecting left / right rear speaker. Select a multi-room in Settings Select Sound Bar + Surround [Samsung] Select HW-J7500R Select Soundbar Backward left shift Rear left speaker selection Backward-right-movement Rear right speaker selection Check if each sound is output in the speaker test Select "OK" after speaker test (complete) When adjusting the volume of the TV remote control, the multi-room link is displayed Firmware upgrade method Select Menu → Customer Support → "Software Update" → "Update to the Internet" If you click "Select", it will be automatically downloaded (if the current TV version is final, there is no new version The phrase appears and does not proceed) "Would you like to upgrade?
