It can't be deleted after recording the TV broadcast.

Expected cause It can be caused by damage to the recorded file (content)

  • Method of action If it is caused by damage to the recorded file (content), it is recommended to delete it manually as a symptom that occurs when the file (content) is damaged. ■ How to delete recorded files
    1.Smart Hub → Run AllShare Play
    2. Run recorded TV
    3. Select a simple remote control menu on the page
    4. Select Delete (
    ※ Not deleted immediately)
    5. Select content to delete
    6. Delete is activated on the right and deleted when pressed.
