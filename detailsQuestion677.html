Function guide Smart Hub application unlock guide [2014 product standard]
① After selecting Apps, press the remote control check button for 2 seconds
② Select Lock/Unlock
③ Enter password (initial value 0000)
④ Check the square (press the OK button)
⑤ Press Save at the Bottom
⑥ Unlocked apps disappear in lock shape [Note] Guide on how to set lock Please proceed with the same lock setting method

  • The system password input window also appears the same. An application that does not have a lock setting item is an application that does not have a lock function. [2012 product standard]
    1.Run Smart Hub → Move the application to unlock.
    2. Click the Easy menu on the remote control and select "Unlock" in the pop-up window.
    3. Enter the password you entered when setting the lock.
    4. Completion popup occurs.
    5. The unlocked application has no lock shape at the bottom left of the application.
