Integrated remote control / smart remote control voice recognition is not working.

Expected cause When ordering voice recognition with the remote control, it must operate while pressing the "voice recognition" button, and if the remote control battery is insufficient, it may not work

  • Method of action Check voice recognition settings You need to check the voice recognition settings to use the voice recognition function. Please change the voice recognition function to On and use it. ■ Voice recognition utterance when watching TV
    1.“High TV power on” or “Smart TV power on” when initial power is turned on
    2. If you do “High TV” or “Smart TV,” it appears at the bottom of the screen as follows. (The default is high TV ) ■ How to turn on the voice recognition function Menu → Select "System" System → Select "Voice and Motion Recognition" Voice and Motion Recognition → Select "Voice Recognition" Voice recognition → Select "on" How to use remote control voice recognition When the voice recognition function is operated using the smart remote control, voice recognition is performed only when a command is used while pressing the “voice recognition” button on the remote control. Please use the voice recognition function while pressing the voice recognition button in the center of the remote control. ■ Voice recognition button location of remote control Press the voice recognition button on the smart remote control and give a command. 2012 2013 2014 Check remote control battery replacement Most of the symptoms that the remote control does not work are caused by the lack of remaining battery capacity of the remote control. The battery of the remote control provided at the time of purchase is naturally discharged in the distribution process, so it is difficult to use it as much as expected, and the remote control battery must be replaced with a new one. In the case of products sold as cheap bundled products, the usage time is much shorter than that of genuine batteries, so it is recommended to use genuine batteries packaged in pairs. When replacing the remote control battery, if the battery used by another product is replaced instead of replaced in a pair, it may not work normally on the TV due to lack of battery voltage. Unlike remote controls used in the past, remote controls on smart TVs are equipped with smart functions and have high power consumption, so even batteries that operated normally in other products must be replaced when replacing remote control batteries. ■ Recommendations for battery replacement of remote control A pair of remote control batteries (recommended) Remote control battery different pair ■ If it is the same symptom after replacement
    1.Battery polarity needs to be checked
    2. It is necessary to check whether the remote control mode is set to TV
    3. It is necessary to check whether it is caused by a three-wavelength fluorescent lamp
