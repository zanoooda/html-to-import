Please tell me the function of soccer and how to watch the broadcast.

Soccer mode description Soccer mode usage environment
1

  • It can only be used in digital broadcasting through RF. (Excluding wired and cable)
    2. A separate recording device (HDD) is required to use the highlight function.
    3. Highlights only work in soccer games. (It cannot be operated in other programs)
    4. The highlight video is based on the commentator's voice, shout, and background color (green). Soccer mode operation function
    1.Control bar: Shows control functions to the recording function.
    2. Zoom in: Zoom in on the selection screen from the 9 split screen to the full screen.
    3. Automatic Highlights: Start recording automatically when connecting external hard drives. (No menu button)
    4. Manual Highlights: You can select and view highlights recorded during recording.
    5. Highlight Player: Shows the recorded highlight video. Soccer mode execution method
    1.You can enjoy Soccer Mode by running it with the remote control.
    2. If you press the zoom button, it will appear in 9 divisions, and if you press the zoom button again, you can zoom in to the entire screen of the selection part.
    3. You can record the video on the air right away, and you can watch this broadcast/highlight at the same time.
    4. Re-viewing highlight scenes is only available in the recording file.
