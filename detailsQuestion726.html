I can't output TV voice through the home theater speaker.

Anynet+, ARC Configuration In order to listen to the sound of V through the home theater, you need to set Anynet+ and ARC to use it

  • Please check the Anynet+ and ARC settings of the TV and home theater before using them. Menu → Select "Aeynet+ (HDMI-CEC)" Select Aeynet+ (HDMI-CEC) → "On" Menu → Select "Settings" Settings → "Sound" → "Audio Return Channel" → Select "Automatic" Check the connection with the ARC terminal The audio return channel (ARC) must be connected to the HDMI terminal of the home theater and the HAMI (ARC marked terminal) terminal of the TV to hear voice in both directions. Please check if the home theater is connected to the HDMI 2 (ARC) terminal of the TV and use it. Connect to HDMI terminal of home theater Connect to HDMI (ARC) terminal of TV Check HDMI cable version It is necessary to check whether the HDMI cable is a cable marked with HIGH SPEED. In order to use the audio return channel (ARC) function, HDMI cables with HIGH SPEED specifications must be used for normal use. Please check the connection cable and use it. Check if HIGH SPEED is marked.
