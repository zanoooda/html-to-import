The format phrase appears when recording.

Expected cause The format method may occur differently from the PC environment

  • Method of action The format supported when recording TV broadcasting is XFS. For reference, PC is Windows-based OS, so it supports FAT16, FAT32, NTFS format, and TV is Linux-based OS, so it supports XFS format. After all, TV and PC are problems caused by different OSs (However, it does not matter when playing) When playing, it means playing a file already stored on a PC or other external device on a TV.
    ※ Caution : When formatting, all previous data will be deleted (We must back up )
