Broadcasting signal check guide If the color looks red only on a specific channel, it may occur as a broadcast signal abnormality, not a product abnormality

  • This is because if there is an abnormality in the product, the same color spreads on all channels. In this case, please contact the broadcaster to check the broadcast signal [wired, cable, skylife, IPTV, etc.]. Self-diagnosis method If the screen looks red, it may occur due to a broadcast signal problem, TV abnormality, or abnormality of the product can be checked through the image quality test function. How to check image quality test Select [Menu → Customer Support → Self Diagnosis → Image Quality Test].
    1.Press the home button of the remote control
    2. Move lower left [Settings]
    3. Menu → Move customer support
    4. Select [Self Diagnosis]
    5. Select [Start Screen Test]
    6. Check the sample image shown on the TV During the image quality test, the high-definition image built into the TV is displayed on the screen. If the color of the high-definition image does not look red and can be clearly seen, , it is a symptom caused by a broadcast signal problem, not a product problem. In this case, please contact the broadcaster to check the broadcast signal [wired, cable, skylife, IPTV, etc.]. If the color looks red on the high-definition image displayed, , it may occur due to circuit problems and you need to get an inspection. How to upgrade the latest version of firmware In this case, you need to upgrade to the latest version of the software, and if the same symptoms occur after the upgrade, please contact the broadcaster for [wired, cable, skylife, IPTV, etc.] set-top box or broadcast signal inspection. There are two ways to upgrade: USB or network. If you upgrade to USB Access to Samsung Electronics homepage (http://www.samsung.com/sec/) → Customer support → Model search → Download → Firmware → Download firmware file → Release compression and save it in USB memory → Connect USB TV to upgrade. Upgrade while connected to the network You can update directly to the network without a separate USB connection. However, it can be updated only when there is an update of the higher version than the current version. [Installation Method] menu → Customer Support → Software Update → Select Update Now.
    1.Press the home button of the remote control
    2. Move lower left [Settings]
    3. Menu → Customer Support → Software update move
    4. Select [Update now]
