I can't record TV broadcasts (PVR)

Expected cause It may occur due to performance constraints (memory, HDD) of the external device

  • There may be no broadcast signal or it may occur under external input conditions. Method of action Check external device recognition The recording function is a function using an external device, and the recording function cannot be used unless the external HDD is recognized. It may occur if the TV USB connection terminal is not connected to the marked part with HDD, and please refer to the product manual for external HDD support. [Note] USB external hard disk (HDD) of 5,400 rpm or more is recommended for recording devices. USB memory sticks and RAID type USB external hard drives are not supported. Check the reception status of DTV broadcasting The recording function allows you to record only DTV reception after directly receiving the antenna cable If the DTV broadcast signal is unstable, it cannot be recorded.
