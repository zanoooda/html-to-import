The screen does not come out when connecting the PC.

Expected cause It may be caused by cable failure and dual display setting not working

  • Method of action Check cable connection location and status - Make sure you do not use an external graphics card. Confirm built-in and external connection External graphics card If the model is equipped with External type You have to connect to the part. The screen appears. Built-in type If there is a connection line, please connect it to the external type. Check dual monitor settings - Inquiries may occur when setting the display to watch PC images on the TV and monitor at the same time. ■ In case of desktop Desktop Dual Monitor Settings Because the procedure and method vary depending on the manufacturer and model of the graphics card, search for the manufacturer you purchased or the model name of the graphics card Multiple Display Settings for Graphic Options Do it. ■ In case of laptop
    ※ Each laptop manufacturer has a different combination key through the FN key. The above is based on Samsung laptop, and please contact the laptop manufacturer where you purchased the combination key. Check Editing External Input Check Editing External Input
