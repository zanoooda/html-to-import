Target model - 2011 3D support model Method of action Check if the Bluetooth of the TV works normally on the mobile phone

  • Main menu of mobile phone → Configuration → Wireless and Network → Bluetooth Settings → Check if there is "DTVBluetooth" at the bottom. The image below is based on Galaxy S2, and the entry method may vary from mobile phone to manufacturer.
    1.Run [Main Menu] on mobile phone wallpaper
    2. Run [CONFIGURATION] in the main menu
    3. Enter Wireless and Network
    4. Enter [Bluetooth Settings]
    5. Check if [DTVBluetooth] appears in the Bluetooth device search, and if it appears, TV Bluetooth works normally.
