Usage guide When connecting a USB ODD device, it can be used only when the spec is compatible between the TV and the ODD device

  • ODD interface : A / V connectivity (connectivity) function must be supported. TV cannot be connected if A / V connectivity is not supported on ODD device. Connection : Only TV USB (HDD) terminals can be used. - The USB (HDD) terminal uses 5V 1.0A spec. - Check voltage / power spec when purchasing ODD device Encoding: The codec of the ODD device and the codec supported by the TV must match. [ AV Connectivity ] What is the function? Ordinary ODD refers to a function that cannot be used by connecting to a TV, but USB flash memory such as a TV can be used like usable USB memory. How to use the AV connectivity function
    1.If you touch the open/close button twice within 1.5 seconds while the ODD tray is open, it switches to AV system support mode and the LED remains lit.
    2. After that, whenever the open / close button is touched twice, ODD is repeatedly switched to normal ODD mode and AV system support mode.
