Mouse setting method It may occur because the wireless mouse selection is not changed, and the mouse selection must be changed

  • METHOD FOR SETTING WIRELESS MOUSE PEARING
    1.Click the menu on the remote control - System - Device Manager → Mouse Settings → Select Add Bluetooth Mouse → Power ON on the back of the mouse → Pairing button.
    2. Run research → Bluetooth mouse is recognized.
    3. Select remote control - select pairing and connection → Bluetooth mouse connection.
    4. Bluetooth Mouse Device Enabled - Completed. Menu - System - Device Manager Mouse setting Add Bluetooth Mouse Mouse rear power ON → Press pairing button Research execution Bluetooth mouse recognized Select Remote Control - Select Pairing and Connection Bluetooth mouse connected Bluetooth Mouse Device Enabled - Completed
