I want to know how to set the TV screen to be suitable for the game.

Usage guide - When you connect a video game console such as PlayStation or Xbox, you can set the TV screen to be suitable for the game

  • You can't use it while watching TV. Connect the video game console and turn on the feature. If you turn on the function before connecting, the image quality may decrease slightly. The screen may shake a little. When using game mode, screen mode is set to standard screen and sound mode is automatically set to watch movies. ☞ Entry method Menu → System → General Settings → Turn off/on Game Mode ☞ Game mode activation & deactivation condition - Game mode: User selection specification when using game console connected - Check the current external input conditions and specifications disable : When watching TV (RF screen), when setting up PC for external input editing in HDMI mode Activation: All external inputs except the deactivation condition
    ※ PC settings can be set to PC from external input → HDMI → simple menu → external input editing.
