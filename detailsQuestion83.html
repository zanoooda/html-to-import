Usage guide When you connect a video game console such as PlayStation or Xbox, you can set the TV screen to be suitable for the game

  • - Not available while watching TV. - Connect the video game console and turn on the function. If you turn on the function before connecting, the image quality may decrease slightly. - The screen may shake a little. - When using game mode, screen mode is set to standard screen and sound mode is automatically set to watch movies. Method of action Menu → System → General Settings → Turn off / turn on game mode
