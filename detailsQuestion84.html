Please let me know how to test HDMI cable for TV products in 2020.

Usage guide If malfunction symptoms such as blinking or power off occur while connecting the TV and external devices to HDMI, you can check for abnormalities in the connected HDMI cable using the HDMI cable test function built into the TV

  • [Note] This feature is only available in the 19-year TV software version 1303.7 or higher. If there is no HDMI cable test function, please update to the latest version software and use it. If you connect a TV and an external device (set-top box, game console, etc.) with an HDMI cable, it is recommended to use a genuine cable. If the certified HDMI cable is not used, malfunction symptoms may occur due to communication errors. The genuine HDMI cable is printed or imprinted on the cable with Ultral High Speed HDMI, and if there is no corresponding phrase, please replace it with a genuine HDMI cable with an uncertified cable. [20 HDMI Cable Test Method] Remote control home button → Settings → Menu → Customer Support → Self Diagnosis → HDMI Problem Inspection Press the remote control home button Move [Settings] at the bottom left Customer support → Move device care [Move] by self-diagnosis Select [Video Test] Select Video Test [Check] Select an external device to diagnose Do you see the connected screen without any problems [Yes or No] selection Completion of checking connected device problem [No selection screen due to connected product problem] HDMI problem diagnosis [Test start] Cable test in progress (takes about 2 minutes) If there is a problem with the HDMI cable, please replace the connected HDMI cable with a genuine cable and use it.
