Usage guide [2017 Integrated Remote Control Setting] 2017 M5500 ↑ Support

  • (Excluding MU6190) The basic concept of TV functions is to change 'channel' and 'volume'.
    1.Remote control home key → external input
    2. Select external input → integrated remote control setting
    3. Add new device
    4. Set-top box (Select external device to connect)
    5. STB and TV connect directly to TV
    6. Area-of-residence
    7. Service provider setting (select cable company)
    8. Select the connected HDMI mode
    9. Set-top box control test (channel up / down)
    10. Select “Yes” if the channel has changed
    11. Setup complete. [Initialization Method] If you return the setting to malfunction or shipment conditions, initialize applied to the product It can be restored under the condition of factory shipment. [Setting Method] Press the home button of the remote control → Settings → Menu → Customer Support → Self Diagnosis → Initialization → Password 0000 → Initialization completed after rebooting Press the remote control home button Move [Settings] at the bottom left Menu → Move customer support Select [Self Diagnosis] Select [Initialization] Password [0000] Enter Select [Initialization] [Check] Reboot when selected and initialization completed
