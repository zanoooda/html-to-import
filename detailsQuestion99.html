Setting method [How to run Multi View in SmartThings ]
①' SmartThings ' Run app - Please run the 'SmartThings' app


  • ②' Registered TV Choose. - Please select the TV from the product card on the home screen.
    ③ Selection - Press (More View button).
    ④' Screen Mirroring (Smart View) ' Selection - Please select 'Smart View'.
    ※ Support target model (Multi View) - 2020 TV: UT8500 or higher / Q series model, lifestyle model (excluding The SERO) - 2021 TV: UA9000 or higher / Q series model, lifestyle model (excluding The SERO) * 8K: Maximum 4 divisions / 4K: Maximum 2 divisions [Tips.] Smart View vs Multi View 1) Smart View ( 혹은 Screen Mirroring ) : The screen of the mobile phone Function to view the screen loudly by sending it to dongle or TV 2) Multi View : With the mobile phone and TV connected through the Smart View function, The ability to view the screen of a mobile phone and TV content at the same time
    ⑤' Start now. Selection - In the app notification window, press 'Start now' to start screen mirroring.
    ⑥ Multiview screen of TV - You can select the split screen with the direction key (◁, ▷) of the TV remote control.
    ※ If the TV does not support Multi View, only the mobile phone screen appears when Smart View is executed. Multi View is supported Multi View is not supported [Multi View Mirroring Setting Method ]
    ①' Top bar of screen' Lowering - Please lower the notification window on your cell phone.
    ②' Smart View Select the icon. - Press 'Smart View' on the Quick Run button.
    ③ TV model Selection - Mirror your phone screen or select a TV model to play content.
    ④' Floating icon Choose. - When the connection with the TV is completed, the 'Smart View' icon-shaped floating icon appears on the screen of the mobile phone. Please press the floating icon on the main screen.
    ※ If you do not see the Smart View floating icon, check the Smart View settings.
    ⑤' Multi View Mirroring Turn it on/off. - Push the 'Multi View Mirroring' switch button to watch mobile phone and TV content together on TV.
    ⑥ Multi View Screen Size Set - You can set the size of the Multi View screen. Please press the 'screen size' button.
    ⑦ Multi View Screen Location Set - One screen can be viewed as a full screen and the other screen can be viewed as a small screen on the full screen. Press the 'Screen Location' button to select the location of the small screen.
    ⑧ Multi View screen change - You can change the currently selected screen to another screen. The sound of the selected screen is output. Please press the 'Change screen' button.
